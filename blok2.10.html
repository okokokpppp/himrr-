Водород

H · Z = 1 · Период 1 · Группа I A / VII A

1. Положение в Периодической системе. Особенности элемента

Водород занимает уникальное положение в Периодической системе — его помещают либо в группу I A (сходство с щелочными металлами: конфигурация 1s¹, образует H⁺ — протон), либо в группу VII A (сходство с галогенами: принимает электрон, образует H⁻ — гидрид-ион). Правильнее рассматривать водород как уникальный элемент, не принадлежащий ни к одной из этих групп.

Электронная конфигурация: 1s¹. Ковалентный радиус = 0,037 нм — наименьший среди всех элементов. Потенциал ионизации I₁ = 1312 кДж/моль — высокий, сопоставим с галогенами. Электроотрицательность по Полингу = 2,20. Водород — самый распространённый элемент во Вселенной (≈75% по массе) и третий по распространённости на Земле (после O и Si), преимущественно в составе воды и органических соединений.

Изотопы водорода: протий ¹H (99,985%), дейтерий ²H (D, 0,015%), тритий ³H (T, радиоактивный, T₁/₂ = 12,3 лет). Тяжёлая вода D₂O: Tпл = 3,82°C, Tкип = 101,4°C, ρ = 1,105 г/мл. При употреблении в больших количествах D₂O токсична — замещает H₂O в биохимических реакциях и нарушает метаболизм.

2. Физические свойства

Водород H₂ — бесцветный газ без запаха и вкуса, наилегчайший из всех газов: M = 2,016 г/моль, ρ = 0,0899 г/л (н.у.) — в 14,5 раза легче воздуха. Tкип = −252,9°C (20,3 К), Tпл = −259,1°C (14,0 К) — вторая по низкости температура кипения после He. Теплота сгорания — наивысшая среди всех простых веществ: 142 МДж/кг (для сравнения: метан — 55 МДж/кг, бензин — 46 МДж/кг). Растворимость в воде мала: 1,8 мл/100 мл H₂O при 20°C. Хорошо растворяется в палладии — Pd поглощает до 900 объёмов H₂.

Молекула H₂ — неполярная, связь σ, длина 0,074 нм, энергия связи 436 кДж/моль — наиболее прочная из одинарных ковалентных связей. Именно высокая прочность H–H обусловливает относительную инертность H₂ при комнатной температуре.

3. Химические свойства

Водород — активный восстановитель при нагревании. В реакциях может выступать и восстановителем (с большинством неметаллов и оксидов металлов), и окислителем (с активными металлами, образуя гидриды со степенью окисления H = −1).

Взаимодействие с галогенами:

H₂ + F₂ → 2HF (взрывообразно даже в темноте при −250°C)

H₂ + Cl₂ → 2HCl (цепная реакция на свету или при поджигании)

H₂ + Br₂ → 2HBr (при нагревании)

H₂ + I₂ ⇌ 2HI (обратимо при высоких температурах)

Взаимодействие с кислородом:

2H₂ + O₂ → 2H₂O (ΔH = −483,6 кДж/моль)

Смесь 2H₂ : 1O₂ называется «гремучим газом» — взрывается при поджигании, от искры или при нагревании. Водород горит в воздухе почти невидимым бледно-голубым пламенем — проверять наличие водорода поджиганием нужно только в небольших количествах, убедившись в отсутствии большого скопления газа.

Взаимодействие с неметаллами:

N₂ + 3H₂ ⇌ 2NH₃ (Fe-катализатор, T = 450°C, P = 150–300 атм — синтез аммиака)

H₂ + S → H₂S (при нагревании, T > 300°C)

C + 2H₂ → CH₄ (гидрирование кокса, T = 1000°C)

Взаимодействие с металлами (H₂ — окислитель, образуются гидриды):

H₂ + 2Na → 2NaH (T > 200°C)

H₂ + Ca → CaH₂ (T = 300–400°C)

H₂ + 2Li → 2LiH

Восстановление оксидов металлов:

H₂ + CuO → Cu + H₂O (при нагревании)

3H₂ + Fe₂O₃ → 2Fe + 3H₂O (при высоких температурах)

3H₂ + WO₃ → W + 3H₂O

H₂ + NiO → Ni + H₂O

Водород не восстанавливает оксиды активных металлов (Na₂O, CaO, Al₂O₃) — они термодинамически слишком устойчивы.

Гидрирование органических соединений (с катализатором Ni или Pt):

жир (масло) + nH₂ → твёрдый жир (маргарин) (T = 180–200°C, кат. Ni)

4. Получение

Промышленные методы:

CH₄ + H₂O → CO + 3H₂ (паровой риформинг метана, T = 800–900°C, Ni-катализатор — основной метод, >95% мирового производства H₂)

CO + H₂O → CO₂ + H₂ (конверсия CO, T = 400°C, Fe₂O₃/Cr₂O₃ — доочистка)

C + H₂O → CO + H₂ (газификация угля, T > 1000°C)

2H₂O → 2H₂ + O₂ (электролиз воды — чистый H₂, но энергозатратно)

Лабораторные методы:

Zn + H₂SO₄(разб.) → ZnSO₄ + H₂↑ (основной лабораторный способ — аппарат Киппа)

Fe + 2HCl → FeCl₂ + H₂↑

2Na + 2H₂O → 2NaOH + H₂↑

Ca + 2H₂O → Ca(OH)₂ + H₂↑

2Al + 2NaOH + 6H₂O → 2Na[Al(OH)₄] + 3H₂↑

Si + 2NaOH + H₂O → Na₂SiO₃ + 2H₂↑

NaH + H₂O → NaOH + H₂↑ (гидролиз гидридов)

Концентрированная H₂SO₄ и HNO₃ (как конц., так и разб.) с большинством металлов дают не H₂, а SO₂, H₂S, NO или NO₂. Водород при реакции с кислотами-окислителями не выделяется.

5. Применение

Синтез аммиака (процесс Габера–Боша) — крупнейший промышленный потребитель H₂; синтез HCl; гидрирование жиров (производство маргарина, кат. Ni); гидроочистка нефтепродуктов (удаление S, N, O-примесей); восстановление металлов из оксидов (W, Mo, Co); атомно-водородная сварка (T ≈ 4000°C — атомарный водород при рекомбинации 2H· → H₂ выделяет огромную энергию локально); охладитель в турбогенераторах; водородное топливо — топливные элементы и ракетное топливо (жидкий H₂ + жидкий O₂ = наиболее энергоёмкая топливная пара).
