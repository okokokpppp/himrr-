Кислоты

1. Определение и классификация

Кислоты — электролиты, диссоциирующие в водном растворе с отщеплением катионов водорода H⁺ (точнее — ионов гидроксония H₃O⁺). Общая формула: HₙA, где A — кислотный остаток, n — основность кислоты (число атомов H, способных замещаться на металл).

По составу кислоты делятся на бескислородные и кислородсодержащие. Бескислородные — бинарные соединения водорода с неметаллом: HF, HCl, HBr, HI, H₂S, HCN. Кислородсодержащие — содержат кислород в кислотном остатке: H₂SO₄, HNO₃, H₃PO₄, H₂CO₃, HClO₄, HMnO₄ и другие.

По основности (числу атомов H, способных замещаться): одноосновные — HCl, HNO₃, HF, CH₃COOH (один H⁺); двухосновные — H₂SO₄, H₂S, H₂CO₃, H₂SiO₃ (два H⁺); трёхосновные — H₃PO₄, H₃BO₃ (три H⁺).

По силе (степени диссоциации в разбавленном растворе): сильные — полностью диссоциируют в разбавленных растворах (HCl, HBr, HI, HNO₃, H₂SO₄, HClO₄, HClO₃, HMnO₄); слабые — диссоциируют частично (HF, H₂S, H₂CO₃, H₂SiO₃, H₃PO₄, HNO₂, CH₃COOH). H₂SO₄ — сильная по первой ступени и умеренно сильная по второй (Ka₂ = 0,012).

По летучести: летучие — легко улетучиваются при нагревании (HCl, HBr, HI, HNO₃, CH₃COOH); нелетучие — не улетучиваются (H₂SO₄, H₃PO₄, H₂SiO₃). Это важно при вытеснении кислот из солей.

Правило вытеснения кислот из солей: более сильная кислота вытесняет более слабую из её соли. Нелетучая кислота вытесняет летучую при нагревании даже если она слабее: H₂SO₄(конц.) + NaCl → NaHSO₄ + HCl↑ (HCl сильнее H₂SO₄ по первой ступени, но улетучивается). Кроме того, нерастворимая кислота выпадает в осадок: Na₂SiO₃ + 2HCl → H₂SiO₃↓ + 2NaCl.

2. Общие химические свойства кислот

2.1. Реакция с металлами

С разбавленными кислотами (не окислителями — HCl, H₂SO₄ разб., HBr, HI) реагируют металлы, стоящие левее водорода в ряду напряжений, с выделением H₂:

Zn + 2HCl → ZnCl₂ + H₂↑

Fe + H₂SO₄(разб.) → FeSO₄ + H₂↑

Mg + 2HCl → MgCl₂ + H₂↑

2Al + 6HCl → 2AlCl₃ + 3H₂↑

С кислотами-окислителями (HNO₃, конц. H₂SO₄) водород не выделяется — реакции рассмотрены в отдельных разделах.

Fe, Al, Cr пассивируются концентрированными HNO₃ и H₂SO₄ при обычных условиях — на поверхности образуется плотная оксидная плёнка. Именно поэтому концентрированную HNO₃ перевозят в стальных цистернах. Au и Pt не реагируют ни с одной кислотой в отдельности — только с «царской водкой» (3HCl + HNO₃).

2.2. Реакция с основными и амфотерными оксидами

CuO + H₂SO₄ → CuSO₄ + H₂O

Fe₂O₃ + 6HCl → 2FeCl₃ + 3H₂O

Al₂O₃ + 6HNO₃ → 2Al(NO₃)₃ + 3H₂O

ZnO + H₂SO₄ → ZnSO₄ + H₂O

MgO + 2HCl → MgCl₂ + H₂O

2.3. Реакция с основаниями (нейтрализация)

NaOH + HCl → NaCl + H₂O

Ca(OH)₂ + 2HNO₃ → Ca(NO₃)₂ + 2H₂O

Cu(OH)₂ + H₂SO₄ → CuSO₄ + 2H₂O

Al(OH)₃ + 3HCl → AlCl₃ + 3H₂O

С многоосновными кислотами при недостатке основания образуются кислые соли:

NaOH + H₃PO₄(недост.) → NaH₂PO₄ + H₂O

2NaOH + H₃PO₄ → Na₂HPO₄ + 2H₂O

NaOH + H₂SO₄(недост.) → NaHSO₄ + H₂O

2.4. Реакция с солями

Реакция идёт если образуется осадок, газ или слабый электролит:

Na₂CO₃ + 2HCl → 2NaCl + H₂O + CO₂↑

Na₂S + 2HCl → 2NaCl + H₂S↑

BaCl₂ + H₂SO₄ → BaSO₄↓ + 2HCl

Na₂SiO₃ + 2HCl → H₂SiO₃↓ + 2NaCl

AgNO₃ + HCl → AgCl↓ + HNO₃

CH₃COONa + HCl → CH₃COOH + NaCl (слабая кислота вытесняется сильной)

3. Важнейшие кислоты

3.1. Соляная кислота HCl

Физические свойства: бесцветная жидкость с резким запахом, дымит на воздухе (HCl(г) + H₂O → туман). Максимальная концентрация водного раствора — 38% («концентрированная соляная кислота»), ρ ≈ 1,19 г/см³. Смешивается с водой в любых соотношениях. Сильная одноосновная кислота, полностью диссоциирует: HCl → H⁺ + Cl⁻. Tкип азеотропной смеси (20,2% HCl) = 108,6°C.

Химические свойства:

Все общие свойства сильных кислот. Специфические реакции:

MnO₂ + 4HCl(конц.) → MnCl₂ + Cl₂↑ + 2H₂O (HCl — восстановитель, окисляется до Cl₂)

2KMnO₄ + 16HCl(конц.) → 2KCl + 2MnCl₂ + 5Cl₂↑ + 8H₂O

K₂Cr₂O₇ + 14HCl(конц.) → 2KCl + 2CrCl₃ + 3Cl₂↑ + 7H₂O

3HCl + HNO₃(конц.) → «царская водка» — растворяет Au и Pt:

Au + HNO₃ + 3HCl → AuCl₃ + NO↑ + 2H₂O

Получение:

H₂ + Cl₂ → 2HCl (синтетический метод — основной промышленный)

NaCl + H₂SO₄(конц.) → NaHSO₄ + HCl↑ (умеренный нагрев)

2NaCl + H₂SO₄(конц.) → Na₂SO₄ + 2HCl↑ (сильный нагрев)

NaCl + H₃PO₄(конц.) → NaH₂PO₄ + HCl↑ (при нагревании)

Применение: травление металлов перед нанесением покрытий; производство хлоридов (FeCl₃, ZnCl₂, AlCl₃); синтез ПВХ (через HCl + C₂H₂ → CH₂=CHCl); гидрометаллургия (выщелачивание руд); пищевая промышленность (E507 — регулятор кислотности); производство желатина и клея из костей; медицина (препараты для лечения ахлоргидрии желудка).

3.2. Серная кислота H₂SO₄

Физические свойства: бесцветная маслянистая жидкость, Tпл = 10,4°C, Tкип = 337°C (с частичным разложением), ρ = 1,84 г/см³ (конц.). Смешивается с водой в любых соотношениях с сильным выделением тепла (ΔH = −96 кДж/моль). Правило разбавления: кислоту вливают в воду, а не наоборот — иначе местный перегрев приводит к выбросу. Гигроскопична — поглощает влагу из воздуха и органических веществ («обугливает» их).

Разбавленная H₂SO₄ — типичная сильная кислота (полная диссоциация по первой ступени). Концентрированная H₂SO₄ — качественно иные свойства: сильный окислитель, пассивирует Fe, Al, Cr.

Химические свойства разбавленной H₂SO₄:

Все общие свойства сильных кислот. Специфические:

BaCl₂ + H₂SO₄(разб.) → BaSO₄↓ + 2HCl (качественная реакция на SO₄²⁻)

Ba²⁺ + SO₄²⁻ → BaSO₄↓ (белый осадок, нерастворимый в HNO₃)

Zn + H₂SO₄(разб.) → ZnSO₄ + H₂↑

Fe + H₂SO₄(разб.) → FeSO₄ + H₂↑ (Fe → +2)

Химические свойства концентрированной H₂SO₄:

Как окислитель — реагирует с большинством металлов (кроме Au, Pt, Ir) и с рядом неметаллов, выделяя SO₂ (реже S или H₂S при действии активных восстановителей):

Cu + 2H₂SO₄(конц.) → CuSO₄ + SO₂↑ + 2H₂O

S + 2H₂SO₄(конц.) → 3SO₂↑ + 2H₂O

C + 2H₂SO₄(конц.) → CO₂↑ + 2SO₂↑ + 2H₂O

2P + 5H₂SO₄(конц.) → 2H₃PO₄ + 5SO₂↑

Zn + 2H₂SO₄(конц.) → ZnSO₄ + SO₂↑ + 2H₂O

8Na + 5H₂SO₄(конц.) → 4Na₂SO₄ + H₂S↑ + 4H₂O (очень активный металл → H₂S)

Fe + H₂SO₄(конц., хол.) — пассивация (реакция не идёт)

2Fe + 6H₂SO₄(конц., гор.) → Fe₂(SO₄)₃ + 3SO₂↑ + 6H₂O (при нагревании пассивация снимается)

Как осушитель и водоотнимающее средство:

C₁₂H₂₂O₁₁ + H₂SO₄(конц.) → 12C + 11H₂O + H₂SO₄ (обугливание сахарозы)

HCOOH + H₂SO₄(конц.) → CO↑ + H₂O (дегидратация муравьиной кислоты — лаб. получение CO)

Взаимодействие с солями (вытеснение летучих кислот):

NaCl + H₂SO₄(конц.) → NaHSO₄ + HCl↑

2NaBr + 2H₂SO₄(конц.) → Na₂SO₄ + Br₂↑ + SO₂↑ + 2H₂O (окисляет Br⁻)

8NaI + 5H₂SO₄(конц.) → 4Na₂SO₄ + 4I₂↑ + H₂S↑ + 4H₂O (окисляет I⁻ до I₂, сама восстанавливается до H₂S)

Получение:

Контактный метод (промышленный, 3 стадии):

4FeS₂ + 11O₂ → 2Fe₂O₃ + 8SO₂ (обжиг пирита или сжигание серы)

2SO₂ + O₂ → 2SO₃ (катализатор V₂O₅, T = 450°C)

SO₃ + H₂SO₄(конц.) → H₂S₂O₇ (олеум); H₂S₂O₇ + H₂O → 2H₂SO₄

(SO₃ не поглощают водой напрямую — образуется туман из мелких капель, трудно улавливаемый.)

Применение: производство удобрений (суперфосфат, аммофос — крупнейший потребитель H₂SO₄); нефтепереработка (алкилирование, сернокислотная очистка); производство красителей, взрывчатых веществ (нитрование органики); металлургия (травление стали, выщелачивание руд); аккумуляторы (электролит в свинцово-кислотных аккумуляторах); производство HCl, HF, H₃PO₄ из солей; синтетические волокна и смолы.

3.3. Азотная кислота HNO₃

Физические свойства: бесцветная (чистая) или желтоватая (из-за растворённого NO₂) жидкость с резким запахом, Tкип = 83°C, ρ = 1,51 г/см³ (конц.). Смешивается с водой. Разлагается на свету и при нагревании (желтеет). «Дымящая» HNO₃ — раствор NO₂ в конц. HNO₃. Сильная одноосновная кислота.

Химические свойства:

HNO₃ — сильный окислитель во всех концентрациях. Водород при реакции с металлами никогда не выделяется. Продукт восстановления азота зависит от концентрации кислоты и активности металла:

Концентрированная HNO₃ (с большинством металлов) → NO₂:

Cu + 4HNO₃(конц.) → Cu(NO₃)₂ + 2NO₂↑ + 2H₂O

Ag + 2HNO₃(конц.) → AgNO₃ + NO₂↑ + H₂O

Fe + 6HNO₃(конц., гор.) → Fe(NO₃)₃ + 3NO₂↑ + 3H₂O

Разбавленная HNO₃ (с малоактивными металлами) → NO:

3Cu + 8HNO₃(разб.) → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O

3Fe + 8HNO₃(разб.) → 3Fe(NO₃)₂ + 2NO↑ + 4H₂O (Fe → +2)

Fe(NO₃)₂ + HNO₃(изб.) → Fe(NO₃)₃ + ... (избыток HNO₃ окисляет Fe²⁺ до Fe³⁺)

Очень разбавленная HNO₃ (с активными металлами) → N₂O, N₂, NH₄NO₃:

4Zn + 10HNO₃(оч.разб.) → 4Zn(NO₃)₂ + NH₄NO₃ + 3H₂O

8Al + 30HNO₃(оч.разб.) → 8Al(NO₃)₃ + 3NH₄NO₃ + 9H₂O

Реакция с неметаллами:

S + 6HNO₃(конц.) → H₂SO₄ + 6NO₂↑ + 2H₂O

P + 5HNO₃(конц.) → H₃PO₄ + 5NO₂↑ + H₂O

C + 4HNO₃(конц.) → CO₂↑ + 4NO₂↑ + 2H₂O

Пассивация: Fe, Al, Cr пассивируются конц. HNO₃ при обычных условиях.

Термическое разложение:

4HNO₃ → 4NO₂↑ + O₂↑ + 2H₂O (при нагревании или на свету)

Получение (метод Оствальда, три стадии):

4NH₃ + 5O₂ → 4NO + 6H₂O (Pt–Rh катализатор, T = 850°C)

2NO + O₂ → 2NO₂

3NO₂ + H₂O → 2HNO₃ + NO↑ (NO возвращают в цикл)

Применение: производство азотных удобрений (NH₄NO₃ — аммиачная селитра, крупнейший потребитель); производство взрывчатых веществ (нитроглицерин, тротил, пироксилин — нитрование органики); производство красителей и фармацевтических препаратов; металлургия — растворение металлов, аффинаж драгоценных металлов; «царская водка» (3HCl : 1HNO₃) — растворение Au и Pt.

3.4. Фосфорная кислота H₃PO₄

Физические свойства: бесцветные гигроскопичные кристаллы (Tпл = 42,4°C), в продаже — 85% водный раствор (сиропообразная жидкость), ρ = 1,685 г/см³. Хорошо растворима в воде. Трёхосновная кислота средней силы: Ka₁ = 7,5·10⁻³, Ka₂ = 6,2·10⁻⁸, Ka₃ = 4,8·10⁻¹³ (третья ступень — очень слабая). Не летуча, не окислитель.

Химические свойства:

Все общие свойства кислот средней силы. Образует три ряда солей — дигидрофосфаты, гидрофосфаты и фосфаты:

NaOH + H₃PO₄ → NaH₂PO₄ + H₂O (дигидрофосфат — при соотношении 1:1)

2NaOH + H₃PO₄ → Na₂HPO₄ + 2H₂O (гидрофосфат)

3NaOH + H₃PO₄ → Na₃PO₄ + 3H₂O (фосфат)

Ca(OH)₂ + 2H₃PO₄ → Ca(H₂PO₄)₂ + 2H₂O (суперфосфат)

3Ca(OH)₂ + 2H₃PO₄ → Ca₃(PO₄)₂↓ + 6H₂O

Качественная реакция на фосфат-ион:

3AgNO₃ + H₃PO₄ → Ag₃PO₄↓(жёлтый) + 3HNO₃

3Ag⁺ + PO₄³⁻ → Ag₃PO₄↓ (жёлтый осадок, растворяется в HNO₃ разб.)

Термическое дегидратирование при нагревании:

2H₃PO₄ → H₄P₂O₇ + H₂O (T ≈ 200°C — пирофосфорная кислота)

H₃PO₄ → HPO₃ + H₂O (T > 300°C — метафосфорная кислота)

Получение:

Ca₃(PO₄)₂ + 3H₂SO₄(конц.) → 2H₃PO₄ + 3CaSO₄↓ (экстракционный метод — основной промышленный)

P₂O₅ + 3H₂O → 2H₃PO₄ (термический метод — чистая кислота)

3P + 5HNO₃(конц.) + 2H₂O → 3H₃PO₄ + 5NO↑

Применение: производство фосфорных удобрений (суперфосфат, двойной суперфосфат, аммофос — крупнейший потребитель); производство моющих средств (фосфаты — умягчители воды, строители); пищевая промышленность (E338 — регулятор кислотности в газированных напитках, придаёт кисловатый вкус «Коле»); антикоррозионные покрытия (фосфатирование стали — фосфатный грунт); стоматология (фосфатный цемент); производство огнеупоров и специальных стёкол.

3.5. Угольная кислота H₂CO₃

Физические свойства: существует только в водном растворе — в чистом виде не выделена. Раствор CO₂ в воде: CO₂ + H₂O ⇌ H₂CO₃ (равновесие сильно сдвинуто влево — лишь ≈0,3% CO₂ реально гидратировано в H₂CO₃). Слабая двухосновная кислота: Ka₁ = 4,3·10⁻⁷, Ka₂ = 4,7·10⁻¹¹. Нестойкая — разлагается при нагревании и при уменьшении давления CO₂.

Химические свойства:

H₂CO₃ + 2NaOH → Na₂CO₃ + 2H₂O

H₂CO₃ + NaOH(недост.) → NaHCO₃ + H₂O

H₂CO₃ → H₂O + CO₂↑ (разложение при нагревании)

Как слабая кислота — вытесняется из карбонатов сильными кислотами:

Na₂CO₃ + H₂SO₄ → Na₂SO₄ + H₂O + CO₂↑

CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑

Слабее H₂SiO₃ — CO₂ вытесняет SiO₂ из силикатов при сплавлении, но H₂CO₃ не вытесняет H₂SiO₃ из растворимых силикатов — наоборот:

Na₂SiO₃ + CO₂ + H₂O → H₂SiO₃↓ + Na₂CO₃ (CO₂ вытесняет H₂SiO₃ из раствора)

Угольная кислота слабее кремниевой в растворе — несмотря на то что CO₂ при высоких температурах вытесняется SiO₂ из карбонатов. Это разные условия: термодинамика при высоких температурах и кинетика в растворе при н.у. — не совпадают.

Применение солей H₂CO₃: Na₂CO₃ (кальцинированная сода) — производство стекла, мыла, бумаги, синтетических моющих средств; NaHCO₃ (пищевая сода) — разрыхлитель в выпечке, гашение ожогов кислотой, медицина; CaCO₃ — наполнитель резины и пластмасс, производство стекла, цемента, мел.

3.6. Кремниевая кислота H₂SiO₃

Физические свойства: белый студенистый осадок (гель), практически нерастворима в воде (0,0001 г/100 мл). Слабая двухосновная кислота, слабее угольной в растворе. Нелетучая. При нагревании разлагается:

H₂SiO₃ → SiO₂ + H₂O (T > 500°C)

Химические свойства:

H₂SiO₃ + 2NaOH → Na₂SiO₃ + 2H₂O

Na₂SiO₃ + CO₂ + H₂O → H₂SiO₃↓ + Na₂CO₃ (вытеснение из силикатов)

Na₂SiO₃ + 2HCl → H₂SiO₃↓ + 2NaCl

H₂SiO₃ не реагирует с кислотами (кроме HF), так как нерастворима.

Получение:

Na₂SiO₃ + 2HCl → H₂SiO₃↓ + 2NaCl (осаждение из раствора жидкого стекла)

Na₂SiO₃ + CO₂ + H₂O → H₂SiO₃↓ + Na₂CO₃

Применение: силикагель (SiO₂·nH₂O — осушитель, сорбент, носитель катализатора); производство силикатных красок; основа для получения SiO₂ высокой чистоты. Соль — Na₂SiO₃ (жидкое стекло) — огнезащитные пропитки, канцелярский клей, пропитка бумаги и дерева.

3.7. Сернистая кислота H₂SO₃

Физические свойства: существует только в водном растворе — в свободном виде не выделена. Раствор SO₂ в воде: SO₂ + H₂O ⇌ H₂SO₃ (равновесная реакция). Слабая двухосновная кислота: Ka₁ = 1,5·10⁻²; Ka₂ = 6,3·10⁻⁸. Нестойкая — при нагревании и на воздухе разлагается.

Химические свойства:

Проявляет и восстановительные, и окислительные свойства — S имеет СО = +4 и может как окисляться до +6, так и восстанавливаться до 0 или −2:

H₂SO₃ + 2NaOH → Na₂SO₃ + 2H₂O

H₂SO₃ + NaOH(недост.) → NaHSO₃ + H₂O

2H₂SO₃ + O₂ → 2H₂SO₄ (медленное окисление на воздухе — причина кислотных дождей)

H₂SO₃ + Cl₂ + H₂O → H₂SO₄ + 2HCl (H₂SO₃ — восстановитель)

H₂SO₃ + 2H₂S → 3S↓ + 3H₂O (H₂SO₃ — окислитель)

5H₂SO₃ + 2KMnO₄ → K₂SO₄ + 2MnSO₄ + 3H₂SO₄ + 2H₂O (восстановитель)

Применение солей H₂SO₃: Na₂SO₃, NaHSO₃ — консерванты (E221, E222), антиоксиданты в пищевой промышленности; отбеливание целлюлозы и тканей; Na₂S₂O₅ (пиросульфит натрия) — консервант в виноделии.

3.8. Плавиковая кислота HF

Физические свойства: бесцветная жидкость (безводный HF, Tкип = 19,5°C) или водный раствор. Слабая кислота (Ka = 6,8·10⁻⁴) — единственный из галогеноводородов, не являющийся сильной кислотой. Слабость объясняется исключительно прочной связью H–F и образованием ассоциатов (HF)ₙ за счёт водородных связей. Крайне агрессивна: вызывает глубокие плохо заживающие ожоги — F⁻ проникает в ткани и связывает ионы Ca²⁺ и Mg²⁺, разрушая биоминеральную матрицу костей.

Химические свойства — специфические:

SiO₂ + 4HF → SiF₄↑ + 2H₂O (единственная кислота, растворяющая стекло)

CaF₂ растворяется только в конц. H₂SO₄ — HF слишком слабая для растворения фторита

2HF + 2NaOH → Na₂F₂... нет: HF + NaOH → NaF + H₂O; HF + NaF ⇌ NaHF₂ (кислый фторид)

Образование комплексных фторидов — характерное свойство HF:

SiF₄ + 2HF → H₂[SiF₆] (гексафторкремниевая кислота)

AlF₃ + 3HF → H₃[AlF₆] → 3Na⁺ + [AlF₆]³⁻ (криолит Na₃AlF₆)

Получение:

CaF₂ + H₂SO₄(конц.) → CaSO₄ + 2HF↑

Применение: производство фторполимеров (тефлон — политетрафторэтилен, фторкаучуки); травление стекла и кварца (матирование, нанесение узоров); травление полупроводниковых пластин (удаление SiO₂ в микроэлектронике); производство криолита Na₃AlF₆ для алюминиевой промышленности; катализатор в нефтепереработке (алкилирование).

3.9. Хромовая кислота H₂CrO₄ и двухромовая кислота H₂Cr₂O₇

Физические свойства: в чистом виде — неустойчивые вещества. Существуют преимущественно в виде солей: хроматов (CrO₄²⁻, жёлтые) и дихроматов (Cr₂O₇²⁻, оранжевые). В кислой среде преобладают дихроматы, в щелочной — хроматы: CrO₄²⁻ + H⁺ ⇌ HCrO₄⁻ ⇌ Cr₂O₇²⁻ + H₂O. Сильные кислоты.

Химические свойства — как окислители (Cr⁶⁺ → Cr³⁺):

K₂Cr₂O₇ + 3H₂S + 4H₂SO₄ → Cr₂(SO₄)₃ + K₂SO₄ + 3S↓ + 7H₂O

K₂Cr₂O₇ + 6FeSO₄ + 7H₂SO₄ → Cr₂(SO₄)₃ + K₂SO₄ + 3Fe₂(SO₄)₃ + 7H₂O

K₂Cr₂O₇ + 14HCl(конц.) → 2KCl + 2CrCl₃ + 3Cl₂↑ + 7H₂O

2CrO₃ + 3S → Cr₂O₃ + 3SO₂ (при нагревании — сильное окисление)

Качественные реакции на хромат и дихромат:

2K₂CrO₄ + H₂SO₄ → K₂Cr₂O₇ + K₂SO₄ + H₂O (в кислой среде — оранжевый цвет)

K₂Cr₂O₇ + 2KOH → 2K₂CrO₄ + H₂O (в щелочной среде — жёлтый цвет)

BaCl₂ + K₂CrO₄ → BaCrO₄↓(жёлтый) + 2KCl (качественная реакция на CrO₄²⁻)

Применение: K₂Cr₂O₇ (бихромат калия) — окислитель в аналитической химии и органическом синтезе; дубление кожи (хромовое дубление); K₂CrO₄ — жёлтый пигмент; CrO₃ — хромирование деталей (составная часть электролита для хромовых покрытий).

3.10. Марганцовая кислота HMnO₄

Физические свойства: существует в виде водного раствора (выделение в чистом виде невозможно — очень нестойкая). Наиболее известна в виде соли — KMnO₄ (перманганат калия, «марганцовка»): тёмно-фиолетовые кристаллы, хорошо растворимые в воде (раствор малиново-фиолетового цвета). Сильная кислота. Mn в степени окисления +7 — сильнейший окислитель.

Химические свойства KMnO₄ — как окислитель (зависит от среды):

В кислой среде Mn⁷⁺ → Mn²⁺ (бесцветный раствор MnSO₄):

2KMnO₄ + 5H₂O₂ + 3H₂SO₄ → 2MnSO₄ + K₂SO₄ + 5O₂↑ + 8H₂O

2KMnO₄ + 5Na₂SO₃ + 3H₂SO₄ → 2MnSO₄ + K₂SO₄ + 5Na₂SO₄ + 3H₂O

2KMnO₄ + 16HCl(конц.) → 2KCl + 2MnCl₂ + 5Cl₂↑ + 8H₂O

В нейтральной среде Mn⁷⁺ → Mn⁴⁺ (бурый осадок MnO₂):

2KMnO₄ + 3MnSO₄ + 2H₂O → 5MnO₂↓ + K₂SO₄ + 2H₂SO₄

2KMnO₄ + H₂O + 5Na₂SO₃ → 2MnO₂↓ + K₂SO₄ + 5Na₂SO₄... (упрощённо)

В щелочной среде Mn⁷⁺ → Mn⁶⁺ (зелёный ManganatK₂MnO₄):

2KMnO₄ + Na₂SO₃ + 2KOH → 2K₂MnO₄ + Na₂SO₄ + H₂O

Разложение при нагревании:

2KMnO₄ → K₂MnO₄ + MnO₂ + O₂↑ (T > 200°C — лабораторный способ получения O₂)

Применение: KMnO₄ — антисептик и дезинфектант в медицине; окислитель в аналитической и препаративной химии (перманганатометрия); обеззараживание воды; производство красителей; Mn₂O₇ — редкий лабораторный реагент.

4. Общие способы получения кислот

4.1. Взаимодействие кислотного оксида с водой

SO₃ + H₂O → H₂SO₄

SO₂ + H₂O → H₂SO₃

CO₂ + H₂O ⇌ H₂CO₃

P₂O₅ + 3H₂O → 2H₃PO₄

N₂O₅ + H₂O → 2HNO₃

(SiO₂ с водой не реагирует — H₂SiO₃ этим способом не получают.)

4.2. Вытеснение более слабой или летучей кислоты из соли

CaF₂ + H₂SO₄(конц.) → CaSO₄ + 2HF↑

Na₂CO₃ + H₂SO₄ → Na₂SO₄ + H₂O + CO₂↑

Na₂SiO₃ + 2HCl → H₂SiO₃↓ + 2NaCl

3Ca₃(PO₄)₂ + 10H₂SO₄(конц.) → 6H₃PO₄ + 10CaSO₄↓ (производство H₃PO₄)

4.3. Синтез из элементов (для бескислородных)

H₂ + Cl₂ → 2HCl

H₂ + F₂ → 2HF (самовоспламенение)

H₂ + S ⇌ H₂S (при нагревании, равновесная)

4.4. Окисление неметаллов азотной кислотой

S + 6HNO₃(конц.) → H₂SO₄ + 6NO₂↑ + 2H₂O

P + 5HNO₃(конц.) → H₃PO₄ + 5NO₂↑ + H₂O
