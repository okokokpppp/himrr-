Бинарные соединения

Бинарные соединения — вещества, молекулы или формульные единицы которых образованы атомами ровно двух химических элементов. Общая формула записывается как EₓYᵧ, где E и Y — символы элементов, x и y — стехиометрические коэффициенты. К бинарным соединениям относятся оксиды, гидриды, галогениды, сульфиды, нитриды, карбиды, фосфиды, силициды и ряд других классов. Их изучение составляет основу неорганической химии, поскольку именно бинарные соединения являются исходными продуктами для синтеза более сложных веществ.

Гидриды

1. Общая характеристика и классификация

Гидриды — бинарные соединения элементов с водородом. В зависимости от природы элемента и типа связи водород может иметь разные степени окисления. В гидридах металлов СО(H) = −1 (водород — анион H⁻, более электроотрицательный из двух). В гидридах неметаллов СО(H) = +1 (водород — менее электроотрицательный: HCl, H₂S, NH₃, H₂O). Исключение: H₂O₂ — СО(H) = +1, СО(O) = −1; гидриды бора B₂H₆ — мостиковое строение.

По типу связи гидриды делят на три группы. Ионные гидриды — соединения щелочных и щелочноземельных металлов: NaH, KH, CaH₂, LiH. Содержат ион H⁻ (гидрид-ион). Высокие температуры плавления, в расплаве проводят ток (на аноде выделяется H₂). Ковалентные гидриды — соединения неметаллов: HCl, H₂S, NH₃, PH₃, SiH₄, B₂H₆. Молекулярные соединения. Металлоподобные гидриды — соединения переходных металлов (PdH₀.₇, TiH₂): атомы H в межузлиях металлической решётки, переменный состав, проводники.

2. Ионные гидриды металлов

Физические свойства: белые кристаллические вещества с ионной решёткой, высокими Tпл (NaH — 800°C, CaH₂ — 816°C), плохо растворяются в органических растворителях. Сильные восстановители.

Химические свойства ионных гидридов:

Реакция с водой — бурная, с выделением H₂ (H⁻ отнимает протон от воды):

NaH + H₂O → NaOH + H₂↑

CaH₂ + 2H₂O → Ca(OH)₂ + 2H₂↑ (используется в полевых условиях для получения H₂)

LiH + H₂O → LiOH + H₂↑

Реакция с кислотами — бурная:

NaH + HCl → NaCl + H₂↑

CaH₂ + 2HBr → CaBr₂ + 2H₂↑

Восстановительные свойства при нагревании:

2NaH + O₂ → Na₂O + H₂O (при нагревании)

2LiH + B₂H₆ → 2LiBH₄ (борогидрид лития — мощный восстановитель в органическом синтезе)

4NaH + SiCl₄ → SiH₄ + 4NaCl (получение силана)

LiAlH₄ + 4H₂O → LiOH + Al(OH)₃ + 4H₂↑ (гидролиз алюмогидрида лития)

LiAlH₄ (алюмогидрид лития) и NaBH₄ (борогидрид натрия) — важнейшие восстанавливающие реагенты в органическом синтезе (восстановление карбонильных групп, сложных эфиров, амидов). Хранят строго без воды — реагируют с влагой воздуха.

Получение ионных гидридов:

2Na + H₂ → 2NaH (T > 200°C, прямой синтез)

Ca + H₂ → CaH₂ (T = 300–400°C)

2Li + H₂ → 2LiH (T > 700°C)

2K + H₂ → 2KH (T > 200°C)

Применение: CaH₂ — портативный источник H₂ (военная и спасательная техника, надувные аэростаты); NaH — сильное основание и восстановитель в органическом синтезе; LiAlH₄ и NaBH₄ — восстановители в органическом синтезе; LiH — материал для ядерных реакторов (замедлитель нейтронов) и в составе ядерных боеголовок (термоядерное горючее ⁶LiD).

3. Летучие водородные соединения неметаллов

Летучие водородные соединения неметаллов — ковалентные гидриды, в которых СО(H) = +1. К ним относятся: галогеноводороды HF, HCl, HBr, HI (рассмотрены в разделе «Галогениды»); халькогеноводороды H₂O, H₂S, H₂Se, H₂Te; пниктогены-гидриды NH₃, PH₃, AsH₃, SbH₃; углеводороды CH₄, C₂H₆ и др. (органическая химия); силан SiH₄ и его гомологи.

Закономерности в ряду HₙE: сила кислот возрастает с увеличением атомного номера элемента в группе (связь H–E слабее у более крупных атомов): HF ≪ HCl < HBr < HI; H₂O ≪ H₂S < H₂Se < H₂Te. Аномалии: HF и H₂O имеют аномально высокие температуры кипения из-за водородных связей.

4. Аммиак NH₃ — важнейший гидрид азота

Физические свойства: бесцветный газ с характерным резким запахом, ρ = 0,771 г/л, Tкип = −33,4°C. Очень хорошо растворим в воде (700 объёмов на 1 объём воды при 20°C — максимальная растворимость среди газов!). Водный раствор — нашатырный спирт (10% NH₃). Молекула имеет форму тригональной пирамиды, полярная, сильные водородные связи. Токсичен, ПДК = 20 мг/м³.

Химические свойства NH₃:

Основные свойства (NH₃ — слабое основание, Kb = 1,8·10⁻⁵):

NH₃ + H₂O ⇌ NH₄OH ⇌ NH₄⁺ + OH⁻ (равновесие, щелочная среда)

NH₃ + HCl → NH₄Cl (белый дым — качественная реакция на NH₃)

NH₃ + HNO₃ → NH₄NO₃

NH₃ + H₂SO₄ → (NH₄)₂SO₄ (аммиачная селитра)

2NH₃ + H₂SO₄ → (NH₄)₂SO₄

Восстановительные свойства:

4NH₃ + 3O₂ → 2N₂ + 6H₂O (горение без катализатора)

4NH₃ + 5O₂ → 4NO + 6H₂O (Pt–Rh катализатор, T = 850°C — метод Оствальда)

2NH₃ + 3CuO → 3Cu + N₂ + 3H₂O (восстановление оксида меди)

8NH₃ + 3Cl₂ → N₂ + 6NH₄Cl (при умеренных условиях)

2NH₃ + 3Cl₂(изб.) → N₂ + 6HCl

Реакции с активными металлами:

2Na + 2NH₃(г) → 2NaNH₂ + H₂↑ (T = 300°C, амид натрия)

2K + 2NH₃(ж) → 2KNH₂ + H₂↑ (в жидком аммиаке)

Комплексообразование (лигандные свойства NH₃):

Cu(OH)₂ + 4NH₃ → [Cu(NH₃)₄](OH)₂ (ярко-синий комплекс — качественная реакция на Cu²⁺)

AgCl + 2NH₃ → [Ag(NH₃)₂]Cl (растворение хлорида серебра в аммиаке)

Получение NH₃:

N₂ + 3H₂ ⇌ 2NH₃ (процесс Габера–Боша: Fe-катализатор, T = 450°C, P = 150–300 атм)

Ca₃N₂ + 6H₂O → 3Ca(OH)₂ + 2NH₃↑ (из нитридов)

NH₄Cl + Ca(OH)₂ → CaCl₂ + 2NH₃↑ + 2H₂O (лабораторный способ — нагревание смеси)

NH₄HCO₃ → NH₃↑ + H₂O + CO₂↑ (разложение при нагревании)

Применение: производство азотных удобрений ((NH₄)₂SO₄, NH₄NO₃, NH₄H₂PO₄, карбамид CO(NH₂)₂); производство HNO₃ (через каталитическое окисление до NO); синтез полимеров (нейлон, капрон через капролактам); холодильный агент (NH₃ — один из первых холодагентов, до сих пор используется в промышленных холодильных установках); нашатырный спирт (медицина); производство взрывчатых веществ.

5. Силан SiH₄ и гидриды кремния

Физические свойства SiH₄: бесцветный газ без запаха, ρ = 1,44 г/л, Tкип = −112°C. На воздухе самовоспламеняется. Аналог метана CH₄, но значительно реакционноспособнее.

Химические свойства:

SiH₄ + 2O₂ → SiO₂ + 2H₂O (горение — самовоспламенение на воздухе)

SiH₄ + 2H₂O → SiO₂ + 4H₂↑ (гидролиз — в отличие от CH₄, который не гидролизуется)

SiH₄ + 2NaOH + H₂O → Na₂SiO₃ + 4H₂↑

Получение:

Mg₂Si + 4HCl → 2MgCl₂ + SiH₄↑

SiCl₄ + 4NaH → SiH₄ + 4NaCl (восстановление тетрахлорида кремния)

Применение: осаждение тонких плёнок SiO₂ и Si в производстве полупроводников (CVD-процессы); производство сверхчистого поликремния для солнечных батарей и микроэлектроники.
