Бинарные соединения

Бинарные соединения — вещества, молекулы или формульные единицы которых образованы атомами ровно двух химических элементов. Общая формула записывается как EₓYᵧ, где E и Y — символы элементов, x и y — стехиометрические коэффициенты. К бинарным соединениям относятся оксиды, гидриды, галогениды, сульфиды, нитриды, карбиды, фосфиды, силициды и ряд других классов. Их изучение составляет основу неорганической химии, поскольку именно бинарные соединения являются исходными продуктами для синтеза более сложных веществ.

Нитриды

1. Общая характеристика и классификация

Нитриды — бинарные соединения элементов с азотом, в которых азот имеет степень окисления −3. Общая формула: MeₓNᵧ или EₓNᵧ. Нитриды образуют как металлы (Li₃N, Mg₃N₂, AlN, TiN, Fe₂N), так и неметаллы (NH₃ — нитрид водорода, BN — нитрид бора, Si₃N₄ — нитрид кремния). По типу связи нитриды разделяют на ионные (щелочных и щелочноземельных металлов), ковалентные (BN, Si₃N₄, AlN) и металлоподобные (TiN, ZrN, VN — проводники, твёрдые, тугоплавкие).

2. Химические свойства нитридов

Гидролиз — большинство нитридов активных металлов легко гидролизуются водой с выделением аммиака:

Li₃N + 3H₂O → 3LiOH + NH₃↑

Mg₃N₂ + 6H₂O → 3Mg(OH)₂ + 2NH₃↑

Ca₃N₂ + 6H₂O → 3Ca(OH)₂ + 2NH₃↑

AlN + 3H₂O → Al(OH)₃ + NH₃↑

Al₄C₃ — не нитрид (карбид!); ср.: Mg₃N₂ + 6HCl → 3MgCl₂ + 2NH₄Cl (с кислотой)

Ковалентные нитриды (BN, Si₃N₄) гидролизу практически не поддаются — высокая химическая инертность.

Реакция с кислотами:

Mg₃N₂ + 8HCl → 3MgCl₂ + 2NH₄Cl

AlN + 4HCl → AlCl₃ + NH₄Cl

Li₃N + 4HCl → 3LiCl + NH₄Cl

Реакция с кислородом (окисление при нагревании):

4AlN + 3O₂ → 2Al₂O₃ + 2N₂↑ (T > 700°C)

2BN + 3O₂ → 2B₂O₃ + 2N₂↑ (T > 1000°C для кубического BN)

3. Важнейшие нитриды

Нитрид лития Li₃N — единственный нитрид щелочного металла, образующийся при прямом взаимодействии с N₂ при обычных условиях (другие щелочные металлы с N₂ не реагируют): 6Li + N₂ → 2Li₃N. Красные кристаллы, Tпл = 813°C. Применяется в твердотельных аккумуляторах (литиевый ионный проводник).

Нитрид магния Mg₃N₂ — жёлтый порошок, образуется при горении магния на воздухе одновременно с MgO: 3Mg + N₂ → Mg₃N₂. Бурно реагирует с водой. Применяется для получения чистого NH₃ и в синтезе нитридной керамики.

Нитрид алюминия AlN — белый порошок, Tпл > 2200°C, широкозонный полупроводник (запрещённая зона 6,2 эВ). Уникальное сочетание: высокая теплопроводность (до 320 Вт/(м·К) для монокристаллов) при высоком электрическом сопротивлении. Применение: подложки для мощных электронных компонентов, УФ-светодиоды, пьезоэлектрические MEMS-датчики.

2Al + N₂ → 2AlN (T > 800°C)

Al₄C₃ + 2N₂ → 4AlN + 3C (карботермическое нитрирование)

Нитрид бора BN — структурный аналог углерода. Гексагональный h-BN («белый графит») — слоистая структура, мягкий, электроизолятор, смазочный материал. Кубический c-BN (боразон, нитрид бора кубический) — по твёрдости уступает только алмазу (9,5 по Моосу); устойчив на воздухе до 1400°C (алмаз сгорает при 850°C). Применение c-BN: сверхтвёрдый абразив и режущий инструмент для обработки чёрных металлов (алмаз с железом реагирует — образует карбиды, поэтому для стали используют c-BN).

B₂O₃ + 2NH₃ → 2BN + 3H₂O (синтез h-BN, T = 900°C)

h-BN → c-BN (при T > 1700°C, P > 5 ГПа — аналогично превращению графита в алмаз)

Нитрид кремния Si₃N₄ — белая керамика, Tпл > 1900°C (с разложением). Исключительная термо- и химическая стойкость: не реагирует с кислотами (кроме HF), со щелочами при обычных условиях. Применение: детали двигателей внутреннего сгорания, лопатки газовых турбин, подшипники, огнеупорная керамика, диэлектрическая плёнка в микроэлектронике.

Нитриды переходных металлов — TiN, ZrN, VN — золотисто-жёлтые (TiN), твёрдые (TiN: 9–10 по Моосу), тугоплавкие (Tпл(TiN) = 2950°C), проводящие соединения с металлической связью. Применение TiN: износостойкое покрытие режущих инструментов и деталей (придаёт золотистый цвет), декоративные покрытия, барьерный слой в микроэлектронике. Получение: 2Ti + N₂ → 2TiN (T > 800°C); осаждение из газовой фазы TiCl₄ + N₂ + H₂ → TiN + 4HCl (PVD/CVD в промышленности).

4. Получение и применение нитридов (общие методы)

3Mg + N₂ → Mg₃N₂ (прямой синтез при нагревании)

6Li + N₂ → 2Li₃N (при обычных условиях)

3Ca + N₂ → Ca₃N₂ (T > 400°C)

2Al + N₂ → 2AlN (T > 800°C)

Si₃N₄: 3SiO₂ + 2N₂ + 6C → Si₃N₄ + 6CO (T = 1400°C)

Применение: нитриды активных металлов — реагенты в синтезе аммиака и нитридов других металлов; нитридная керамика (AlN, Si₃N₄, BN) — высокотехнологичные материалы для электроники, авиации, машиностроения; TiN — твёрдые покрытия инструментов; Li₃N — литиевые аккумуляторы нового поколения.
