14. Способы выражения концентрации. Растворы. Кристаллогидраты

14.1. Природа растворения. Растворимость

Растворение — физико-химический процесс, включающий разрушение структуры растворяемого вещества и сольватацию (гидратацию в воде) его частиц растворителем. ΔH_раств = ΔH_решётки + ΔH_гидратации. Если ΔH_гидратации > ΔH_решётки — растворение экзотермично (NaOH, H₂SO₄, CaCl₂); обратное — эндотермично (NH₄NO₃, KNO₃).

Растворимость — максимальная масса вещества (г), растворяющаяся в 100 г растворителя при данной T и P (насыщенный раствор). Классификация: растворимые (Р) > 1 г/100 г H₂O; малорастворимые (М) 0,01–1 г; нерастворимые (Н) < 0,01 г (абсолютно нерастворимых веществ не существует).

Влияние температуры: для большинства твёрдых веществ растворимость растёт с T. Исключения: Ca(OH)₂, Li₂SO₄, Na₂SO₄ (при T > 32,4°C растворимость убывает). Для газов растворимость убывает с T и прямо пропорциональна P (закон Генри: c = K_H · p).

14.2. Способы выражения концентрации

Массовая доля: ω(вещества) = m(вещества)/m(раствора). Выражается в долях или %. Связь с молярной концентрацией: C = (1000 · ρ · ω) / M, где ρ — плотность раствора (г/мл), M — молярная масса (г/моль).

Молярная концентрация: C(B) = n(B) / V(раствора), моль/л. Удобна для расчётов реакций в растворе. При разбавлении: C₁V₁ = C₂V₂.

Моляльность: b(B) = n(B) / m(растворителя), моль/кг. Не зависит от температуры. Применяется в криоскопии и эбуллиоскопии.

Нормальная концентрация: C_н = n(эквивалентов)/V, моль_экв/л. Удобна в кислотно-основных расчётах и ОВР: C_н(кислоты) = C · (число H⁺); C_н(основания) = C · (число OH⁻); C_н(окислителя/восстановителя) = C · (число e⁻).

14.3. Коллигативные свойства. Криоскопия и эбуллиоскопия

Коллигативные свойства зависят от числа частиц растворённого вещества, но не от их природы. Для электролитов вводят изотонический коэффициент i = 1 + α·(n−1), где n — число ионов из формульной единицы, α — степень диссоциации. Повышение температуры кипения: ΔTкип = Ke · b · i (Ke(H₂O) = 0,512 К·кг/моль). Понижение температуры замерзания: ΔTзам = Kk · b · i (Kk(H₂O) = 1,86 К·кг/моль). Осмотическое давление: π = C · R · T.

14.4. Кристаллогидраты

Кристаллогидраты содержат строго определённое число молекул воды кристаллизации, включённых в решётку. Примеры: CuSO₄·5H₂O (медный купорос, синий; безводный CuSO₄ — белый), FeSO₄·7H₂O (железный купорос), Na₂SO₄·10H₂O (глауберова соль), Na₂CO₃·10H₂O (кристаллическая сода), MgSO₄·7H₂O (эпсомит), CaSO₄·2H₂O (гипс → при нагреве до 150°C → CaSO₄·0,5H₂O — алебастр), CoCl₂·6H₂O (красный; безводный CoCl₂ — синий, индикатор влажности).

Расчёт с кристаллогидратами: при растворении кристаллогидрата вода кристаллизации переходит в растворитель. Пример: из 250 г CuSO₄·5H₂O (M = 250 г/моль → 1 моль) получим m(CuSO₄) = 160 г, m(H₂O крист.) = 90 г. При добавлении 410 г воды: m(раствора) = 660 г; ω(CuSO₄) = 160/660 ≈ 24,2%.

Выветривание — самопроизвольная потеря воды кристаллизации при низкой влажности (Na₂CO₃·10H₂O → Na₂CO₃·H₂O). Гигроскопичность — поглощение влаги из воздуха (CaCl₂, P₂O₅, конц. H₂SO₄ используются как осушители).
