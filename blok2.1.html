Бинарные соединения

Бинарные соединения — вещества, молекулы или формульные единицы которых образованы атомами ровно двух химических элементов. Общая формула записывается как EₓYᵧ, где E и Y — символы элементов, x и y — стехиометрические коэффициенты. К бинарным соединениям относятся оксиды, гидриды, галогениды, сульфиды, нитриды, карбиды, фосфиды, силициды и ряд других классов. Их изучение составляет основу неорганической химии, поскольку именно бинарные соединения являются исходными продуктами для синтеза более сложных веществ.

Оксиды

1.1. Классификация оксидов

Оксиды — бинарные соединения элементов с кислородом, в которых кислород имеет степень окисления −2. По способности образовывать соли все оксиды делятся на две группы: солеобразующие и несолеобразующие. Солеобразующие в свою очередь делятся на основные, амфотерные и кислотные.

Основные оксиды образованы металлами в степени окисления +1 и +2 (кроме ZnO, BeO, SnO, PbO): Na₂O, CaO, FeO, CuO. Им соответствуют основания. Амфотерные оксиды образованы металлами в степенях окисления +3 и +4, а также четырьмя исключениями в СО +2 — ZnO, BeO, SnO, PbO: Al₂O₃, Cr₂O₃, ZnO, SnO₂. Реагируют и с кислотами, и со щелочами. Кислотные оксиды образованы неметаллами в любой СО и металлами в СО +5 и выше: SO₂, SO₃, CO₂, P₂O₅, SiO₂, CrO₃, Mn₂O₇. Им соответствуют кислоты. Несолеобразующие оксиды не образуют солей ни с кислотами, ни со щелочами: CO, NO, N₂O, SiO.

Один и тот же элемент образует оксиды разных типов в зависимости от степени окисления. Марганец: MnO (основный) → MnO₂ (амфотерный) → Mn₂O₇ (кислотный). Хром: CrO (основный) → Cr₂O₃ (амфотерный) → CrO₃ (кислотный). Железо: FeO (основный) → Fe₂O₃ (амфотерный, слабо выраженный).

1.2. Основные оксиды

Основные оксиды образованы металлами в степени окисления +1 и +2. Им соответствуют основания. Типичные представители: Li₂O, Na₂O, K₂O (СО +1); MgO, CaO, BaO, FeO, CuO (СО +2). Чем активнее металл — тем более выражены основные свойства его оксида.

Химические свойства основных оксидов:

Реакция с водой — характерна только для оксидов активных металлов (щелочных и щелочноземельных):

Na₂O + H₂O → 2NaOH

CaO + H₂O → Ca(OH)₂ (гашение извести, ΔH = −65 кДж/моль)

BaO + H₂O → Ba(OH)₂

Оксиды малоактивных металлов (CuO, FeO, NiO) с водой не реагируют — соответствующие гидроксиды получают косвенным путём.

Реакция с кислотами — все основные оксиды реагируют с кислотами, образуя соль и воду:

CuO + H₂SO₄ → CuSO₄ + H₂O

FeO + 2HCl → FeCl₂ + H₂O

CaO + 2HNO₃ → Ca(NO₃)₂ + H₂O

MgO + H₂SO₄ → MgSO₄ + H₂O

Реакция с кислотными оксидами — при нагревании или сплавлении:

CaO + CO₂ → CaCO₃

CaO + SiO₂ → CaSiO₃

Na₂O + SO₃ → Na₂SO₄

3CaO + P₂O₅ → Ca₃(PO₄)₂

Реакция с амфотерными оксидами — при сплавлении:

CaO + Al₂O₃ → Ca(AlO₂)₂

Na₂O + Al₂O₃ → 2NaAlO₂

Со щелочами основные оксиды не реагируют — это принципиальный признак, отличающий их от амфотерных.

Физические свойства важнейших основных оксидов:

Оксид натрия Na₂O — белое кристаллическое вещество, Tпл = 1275°C, ρ = 2,27 г/см³. Неустойчив на воздухе — быстро реагирует с CO₂ и H₂O, хранится в инертной атмосфере. Сильный раздражитель кожи и слизистых.

Оксид кальция CaO (негашёная известь) — белое твёрдое вещество, Tпл = 2614°C, ρ = 3,35 г/см³. При взаимодействии с водой локальная температура достигает 300°C. Один из крупнотоннажных продуктов промышленности.

Оксид магния MgO (жжёная магнезия) — белый порошок, Tпл = 2852°C, ρ = 3,58 г/см³. После высокотемпературного прокаливания практически не реагирует с водой.

Оксид железа(II) FeO — чёрный порошок, Tпл = 1377°C, ρ = 5,74 г/см³. Нестехиометрическое соединение (Fe₁₋ₓO). Промежуточный продукт в доменном процессе.

Оксид меди(II) CuO — чёрный порошок, Tпл = 1326°C, ρ = 6,31 г/см³.

Получение основных оксидов:

4Na + O₂ → 2Na₂O (сжигание натрия в недостатке O₂)

2Cu + O₂ → 2CuO (при нагревании)

2Mg + O₂ → 2MgO (сжигание магния)

Cu(OH)₂ → CuO + H₂O (разложение гидроксида при нагревании)

Fe(OH)₂ → FeO + H₂O (без доступа воздуха)

Mg(OH)₂ → MgO + H₂O (T > 350°C)

CaCO₃ → CaO + CO₂ (обжиг известняка, T > 900°C)

FeCO₃ → FeO + CO₂

2Cu(NO₃)₂ → 2CuO + 4NO₂↑ + O₂↑ (T ≈ 300°C)

Fe₂O₃ + CO → 2FeO + CO₂ (восстановление высшего оксида, T = 700–900°C)

Применение: CaO (негашёная известь) — производство цемента, стекла, флюс в металлургии, дезинфекция; MgO — огнеупорная керамика, футеровка печей, медицина (антацид); CuO — пигмент, катализатор, производство Cu и её солей; Fe₃O₄ — сырьё для производства стали, воронение металлических изделий, пигменты.

1.3. Амфотерные оксиды

Амфотерные оксиды образованы металлами в степенях окисления +3 и +4, а также четырьмя исключениями в СО +2: ZnO, BeO, SnO, PbO. Реагируют как с кислотами (проявляя основные свойства), так и со щелочами (проявляя кислотные свойства). Типичные представители: Al₂O₃, ZnO, BeO, Cr₂O₃, SnO₂, PbO₂, SnO, PbO.

Химические свойства амфотерных оксидов:

Реакция с кислотами:

Al₂O₃ + 6HCl → 2AlCl₃ + 3H₂O

ZnO + H₂SO₄ → ZnSO₄ + H₂O

BeO + 2HCl → BeCl₂ + H₂O

Cr₂O₃ + 6HCl → 2CrCl₃ + 3H₂O

SnO + H₂SO₄ → SnSO₄ + H₂O

Реакция со щелочами:

Al₂O₃ + 2NaOH → 2NaAlO₂ + H₂O (при сплавлении)

Al₂O₃ + 2NaOH + 3H₂O → 2Na[Al(OH)₄] (в водном растворе)

ZnO + 2NaOH → Na₂ZnO₂ + H₂O (при сплавлении)

ZnO + 2NaOH + H₂O → Na₂[Zn(OH)₄] (в растворе)

BeO + 2NaOH(конц.) → Na₂BeO₂ + H₂O

Cr₂O₃ + 2NaOH → 2NaCrO₂ + H₂O (при сплавлении)

PbO + 2NaOH → Na₂PbO₂ + H₂O (при сплавлении)

Реакция с основными и кислотными оксидами при сплавлении:

Al₂O₃ + Na₂O → 2NaAlO₂

Al₂O₃ + SO₃ → Al₂(SO₄)₃

α-Al₂O₃ (корунд) химически чрезвычайно инертен — не реагирует ни с кислотами, ни со щелочами при обычных условиях и даже при кипячении, только при сплавлении. γ-Al₂O₃ (активный оксид алюминия) значительно реакционноспособнее. Именно инертность корунда объясняет его применение как абразива и огнеупора.

Физические свойства важнейших амфотерных оксидов:

Оксид алюминия Al₂O₃ (корунд) — белое вещество, Tпл = 2072°C, ρ = 4,0 г/см³, твёрдость 9 по Моосу. Природные окрашенные разновидности: рубин (Cr³⁺ → красный), сапфир (Fe²⁺/Ti⁴⁺ → синий).

Оксид цинка ZnO — белый порошок (при нагревании желтеет обратимо — термохромизм), Tпл = 1975°C, ρ = 5,61 г/см³. Не растворяется в воде.

Оксид хрома(III) Cr₂O₃ — тёмно-зелёный порошок, Tпл = 2435°C, ρ = 5,22 г/см³, твёрдость ≈ 9 по Моосу. Не растворяется в кислотах и щелочах при обычных условиях — только при сплавлении.

Оксид бериллия BeO — белый порошок, Tпл = 2530°C, ρ = 3,01 г/см³. Уникальное сочетание высокой теплопроводности и электрической изоляции. Высокотоксичен.

Оксид олова(II) SnO — чёрный или серый порошок; оксид свинца(II) PbO — жёлтый или красный в зависимости от модификации (глёт и массикот).

Получение амфотерных оксидов:

4Al + 3O₂ → 2Al₂O₃ (T > 800°C)

2Zn + O₂ → 2ZnO

2Al(OH)₃ → Al₂O₃ + 3H₂O (T > 300°C)

Zn(OH)₂ → ZnO + H₂O (T ≈ 200°C)

2Cr(OH)₃ → Cr₂O₃ + 3H₂O

ZnCO₃ → ZnO + CO₂ (T > 300°C)

4Al(NO₃)₃ → 2Al₂O₃ + 12NO₂↑ + 3O₂↑

Применение: Al₂O₃ — абразив (наждачная бумага, шлифовальные круги), огнеупорная керамика, производство Al электролизом расплава в криолите, синтетические рубин и сапфир для лазеров и ювелирного дела; ZnO — белый пигмент (цинковые белила), антисептик (мазь ZnO), вулканизатор каучука; Cr₂O₃ — зелёный пигмент, полировочный порошок (паста ГОИ), производство Cr алюмотермией; BeO — замедлитель нейтронов в ядерных реакторах, подложки для мощных полупроводниковых приборов; PbO — производство свинцового стекла (хрусталь) и свинцовых аккумуляторов.

1.4. Кислотные оксиды

Кислотные оксиды (ангидриды кислот) образованы неметаллами в любой степени окисления, а также металлами в степени окисления +5 и выше. Им соответствуют кислородсодержащие кислоты. Типичные представители из неметаллов: CO₂, SO₂, SO₃, NO₂, N₂O₅, P₂O₅, SiO₂, Cl₂O₇; из металлов в высоких СО: CrO₃, Mn₂O₇, V₂O₅. Чем выше степень окисления центрального атома — тем сильнее кислотные свойства: SO₃ кислотнее SO₂; N₂O₅ кислотнее N₂O₃.

Химические свойства кислотных оксидов:

Реакция с водой — большинство кислотных оксидов реагируют с водой, образуя кислоту:

SO₂ + H₂O → H₂SO₃

SO₃ + H₂O → H₂SO₄

CO₂ + H₂O ⇌ H₂CO₃ (равновесная, Ka₁ = 4,3·10⁻⁷)

N₂O₅ + H₂O → 2HNO₃

P₂O₅ + 3H₂O → 2H₃PO₄

3SO₃ + H₂O → H₂S₃O₁₀ (при растворении в воде образуется олеум при избытке SO₃)

Исключение — SiO₂ с водой не реагирует: кремниевую кислоту H₂SiO₃ получают только косвенно — осаждением из растворов силикатов.

Реакция со щелочами:

CO₂ + 2NaOH → Na₂CO₃ + H₂O

CO₂ + NaOH → NaHCO₃ (при избытке CO₂)

SO₂ + 2NaOH → Na₂SO₃ + H₂O

SO₃ + 2KOH → K₂SO₄ + H₂O

SiO₂ + 2NaOH → Na₂SiO₃ + H₂O (только при сплавлении или кипячении с конц. щёлочью)

P₂O₅ + 6NaOH → 2Na₃PO₄ + 3H₂O

N₂O₅ + 2KOH → 2KNO₃ + H₂O

Реакция с основными оксидами при нагревании:

SO₃ + CaO → CaSO₄

CO₂ + CaO → CaCO₃

SiO₂ + CaO → CaSiO₃

SiO₂ + Na₂O → Na₂SiO₃

P₂O₅ + 3Na₂O → 2Na₃PO₄

Вытеснение более слабого кислотного оксида при сплавлении:

SiO₂ + Na₂CO₃ → Na₂SiO₃ + CO₂↑ (T > 900°C)

SiO₂ + CaCO₃ → CaSiO₃ + CO₂↑

SiO₂ не реагирует с водой и не реагирует ни с одной кислотой, кроме плавиковой: SiO₂ + 4HF → SiF₄↑ + 2H₂O. Именно поэтому концентрированные кислоты хранят в стеклянной посуде, а плавиковую кислоту — только в полиэтиленовой или тефлоновой.

Физические свойства важнейших кислотных оксидов:

Оксид углерода(IV) CO₂ — бесцветный газ без запаха, ρ = 1,977 г/л (тяжелее воздуха), Tкип = −78,5°C (сублимация при атм. давлении — «сухой лёд»). Не горит, не поддерживает горение большинства веществ. Растворяется в воде с образованием слабой угольной кислоты.

Оксид серы(IV) SO₂ — бесцветный газ с резким удушливым запахом, ρ = 2,93 г/л, Tкип = −10°C. Хорошо растворим в воде. Токсичен — ПДК = 10 мг/м³. Один из основных компонентов кислотных дождей.

Оксид серы(VI) SO₃ — бесцветная жидкость (Tкип = 45°C) или белые кристаллы при комнатной температуре, ρ = 1,92 г/см³. Дымит на воздухе, реагируя с влагой. Чрезвычайно агрессивен — вызывает тяжёлые химические ожоги.

Оксид азота(IV) NO₂ — красно-бурый газ с резким запахом, ρ = 1,88 г/л, Tкип = +21,2°C. Токсичен (ПДК = 5 мг/м³). Существует в равновесии с бесцветным димером: 2NO₂ ⇌ N₂O₄ (смещается вправо при охлаждении). С водой диспропорционирует: 3NO₂ + H₂O → 2HNO₃ + NO↑.

Оксид фосфора(V) P₂O₅ — белый порошок (реально тетрамер P₄O₁₀), Tпл = 562°C. Сильнейший осушитель — поглощает воду из любых веществ, включая концентрированные кислоты. Крайне гигроскопичен.

Диоксид кремния SiO₂ (кварц, кремнезём) — бесцветное кристаллическое вещество с атомной ковалентной решёткой, Tпл = 1713°C, ρ = 2,65 г/см³, твёрдость 7 по Моосу. Не проводит ток, прозрачен в видимом и УФ-диапазонах. Химически инертен — не реагирует с водой и кислотами (кроме HF), не реагирует с водой напрямую даже при нагревании. Природные разновидности: кварц, горный хрусталь, аметист, агат, кремень, опал (аморфный).

Получение кислотных оксидов:

S + O₂ → SO₂ (сжигание серы)

4P + 5O₂ → 2P₂O₅

C + O₂ → CO₂ (полное сгорание углерода)

2SO₂ + O₂ → 2SO₃ (катализатор V₂O₅, T = 450°C — контактный метод производства H₂SO₄)

2NO + O₂ → 2NO₂ (при комнатной температуре, без катализатора)

Na₂SO₃ + H₂SO₄ → Na₂SO₄ + H₂O + SO₂↑ (вытеснение слабой кислоты)

CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑ (лабораторный способ получения CO₂)

Cu + 4HNO₃(конц.) → Cu(NO₃)₂ + 2NO₂↑ + 2H₂O

SiO₂ в природе получается в результате геологических процессов. В промышленности — восстановлением кварца в производстве Si (попутный продукт), а также при синтезе стекла.

Применение:

CO₂ — газирование напитков (E290), пожаротушение, сухой лёд для охлаждения, производство соды (Na₂CO₃) и мочевины.

SO₂ — производство H₂SO₄ (промежуточный продукт), отбеливание целлюлозы и тканей, консервант в пищевой промышленности (E220).

SO₃ — производство серной кислоты и олеума, сульфирование органических соединений.

NO₂ — производство HNO₃ по методу Оствальда.

P₂O₅ — лабораторный осушитель, производство фосфорных кислот и удобрений.

SiO₂ — производство стекла всех видов (оконное, бутылочное, оптическое); оптоволоконные кабели (сверхчистый плавлёный кварц); полупроводниковая промышленность (восстановление до кремния: SiO₂ + 2C → Si + 2CO↑, T > 1700°C); огнеупорные материалы и керамика; кварцевые резонаторы и генераторы; наполнители резины и лаков (аморфный SiO₂).

1.5. Несолеобразующие (безразличные) оксиды

Несолеобразующие оксиды — оксиды, которым не соответствует ни одна соль. Не реагируют с водой, кислотами и щелочами при обычных условиях. Образованы неметаллами в промежуточных степенях окисления, при которых ни основные, ни кислотные свойства не проявляются. В программе ОГЭ/ЕГЭ: CO, NO, N₂O, SiO.

Оксид углерода(II) CO — угарный газ

Физические свойства: бесцветный газ без запаха и вкуса, ρ = 1,25 г/л, Tкип = −191,5°C. Практически нерастворим в воде. Горит голубоватым пламенем. Крайне токсичен: связывается с гемоглобином с образованием карбоксигемоглобина (сродство к Hb в 200–250 раз выше, чем у O₂). Смертельная концентрация — 0,1% об. при длительном вдыхании. Главная опасность — полное отсутствие запаха.

Химические свойства:

CO не реагирует с водой, кислотами и щелочами — ему не соответствует никакая соль.

2CO + O₂ → 2CO₂ (горение, голубое пламя)

Fe₂O₃ + 3CO → 2Fe + 3CO₂ (T = 700–900°C, доменный процесс)

FeO + CO → Fe + CO₂

Fe₃O₄ + 4CO → 3Fe + 4CO₂

CuO + CO → Cu + CO₂ (восстановление оксида меди)

NiO + CO → Ni + CO₂ (рафинирование никеля — метод Монда)

CO + Cl₂ → COCl₂ (фосген, активированный уголь — катализатор)

CO + NaOH → HCOONa (при нагревании и давлении — исключение из несолеобразующего поведения)

CO + 3H₂ → CH₄ + H₂O (метанирование, T = 250°C, Ni-катализатор)

CO + 2H₂ → CH₃OH (синтез метанола, T = 250°C, Cu/ZnO-катализатор)

Получение:

2C + O₂ → 2CO (неполное сгорание при недостатке O₂, T > 1000°C)

C + CO₂ → 2CO (реакция Будуара, T > 1000°C)

C + H₂O → CO + H₂ (водяной газ, T = 1000°C)

CH₄ + H₂O → CO + 3H₂ (паровая конверсия метана, T = 800°C, Ni-катализатор)

HCOOH → CO↑ + H₂O (разложение муравьиной кислоты конц. H₂SO₄, лабораторный способ)

H₂C₂O₄ → CO↑ + CO₂↑ + H₂O (разложение щавелевой кислоты конц. H₂SO₄)

Применение: восстановитель в металлургии (доменный процесс, производство Ni, Cu); компонент синтез-газа (CO + H₂) для производства метанола, синтетических углеводородов (реакция Фишера–Тропша) и уксусной кислоты; производство фосгена COCl₂ (поликарбонаты, изоцианаты); топливный газ (водяной газ, генераторный газ).

Оксид азота(II) NO

Физические свойства: бесцветный газ без запаха, ρ = 1,34 г/л, Tкип = −151,8°C. Плохо растворим в воде. Парамагнитен (молекула-радикал с нечётным числом электронов). Умеренно токсичен (ПДК = 5 мг/м³). Эндогенный сосудорасширяющий сигнальный радикал в организме (Нобелевская премия 1998 г.).

Химические свойства:

NO не реагирует с водой, кислотами и щелочами — ему не соответствует никакая соль.

2NO + O₂ → 2NO₂ (быстро, при комнатной температуре, без катализатора)

2NO + Cl₂ → 2NOCl (нитрозилхлорид)

2NO + 2H₂ → N₂ + 2H₂O (при нагревании — каталитическое восстановление)

4NO + 3O₂ + 2H₂O → 4HNO₃ (при избытке кислорода и воды)

NO + O₃ → NO₂ + O₂ (реакция с озоном — разрушение озонового слоя)

Получение:

4NH₃ + 5O₂ → 4NO + 6H₂O (Pt–Rh катализатор, T = 850–900°C — промышленный метод Оствальда)

3Cu + 8HNO₃(разб.) → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O (лабораторный способ)

3Fe + 8HNO₃(разб.) → 3Fe(NO₃)₂ + 2NO↑ + 4H₂O

N₂ + O₂ → 2NO (в грозовых разрядах и электрической дуге, T > 2000°C)

Применение: ключевой промежуточный продукт в производстве азотной кислоты; донаторы NO в медицине (нитроглицерин, нитропруссид натрия) — применяются при стенокардии и гипертоническом кризе.

Оксид азота(I) N₂O — закись азота

Физические свойства: бесцветный газ со слабым приятным запахом, ρ = 1,98 г/л (тяжелее воздуха), Tкип = −88,5°C. Умеренно растворим в воде и хорошо растворим в жирах. Сжижается при давлении 72 атм при 20°C. Оказывает анестезирующее и эйфоризирующее действие при концентрациях 60–80%; при злоупотреблении вызывает необратимые неврологические нарушения (дефицит витамина B₁₂).

Химические свойства:

N₂O не реагирует с водой, кислотами и щелочами — ему не соответствует никакая соль. При нагревании разлагается и поддерживает горение выделяющимся кислородом.

2N₂O → 2N₂ + O₂ (T > 500°C)

Mg + N₂O → MgO + N₂↑ (магний горит в атмосфере N₂O)

C + 2N₂O → CO₂ + 2N₂ (горение углерода в N₂O при нагревании)

H₂ + N₂O → H₂O + N₂ (восстановление водородом при нагревании)

N₂O поддерживает горение — тлеющая лучинка вспыхивает в его атмосфере так же, как в кислороде. Это распространённая ловушка в заданиях ОГЭ/ЕГЭ: поддержание горения не означает солеобразования. N₂O является окислителем при нагревании (выделяет O₂), но соли при этом не образует.

Получение:

NH₄NO₃ → N₂O↑ + 2H₂O (T = 250–270°C; при перегреве выше 300°C — взрывоопасно)

2NH₂OH + (NH₄)₂Cr₂O₇ → N₂O↑ + ... (окисление гидроксиламина — лабораторный способ)

Применение: ингаляционный анестетик в медицине и стоматологии (смесь 50/50 с O₂ — «Энтонокс»); пропеллент в баллончиках со взбитыми сливками (хорошо растворяется в жирах); парниковый газ (потенциал глобального потепления в 298 раз выше CO₂ за 100 лет).

Оксид кремния(II) SiO

Физические свойства: бурый или чёрный аморфный порошок, ρ ≈ 2,2 г/см³. При обычных условиях нестабилен и склонен к диспропорционированию.

Химические свойства:

SiO не реагирует с водой, кислотами и щелочами при обычных условиях — солей не образует. При нагревании диспропорционирует:

2SiO → Si + SiO₂ (при нагревании)

SiO + O₂ → SiO₂ (окисление кислородом при нагревании)

Получение:

SiO₂ + Si → 2SiO (T > 1200°C, в вакууме)

SiO₂ + C → SiO + CO (T > 1500°C)

Применение: встречается как промежуточный продукт в высокотемпературной металлургии кремния; изучается как компонент покрытий в микроэлектронике. В рамках школьной программы рассматривается прежде всего как пример несолеобразующего оксида.

1.6. Пероксиды и надпероксиды

Пероксиды и надпероксиды формально не являются оксидами в классическом смысле, поскольку степень окисления кислорода в них отличается от −2. Тем не менее они встречаются в заданиях ОГЭ/ЕГЭ при изучении химии щелочных и щелочноземельных металлов, а также в темах, связанных с окислительно-восстановительными реакциями и реакциями диспропорционирования.

В пероксидах кислород имеет степень окисления −1 и входит в состав пероксидной группы O₂²⁻. Типичные представители: H₂O₂, Na₂O₂, BaO₂, K₂O₂. В надпероксидах кислород имеет степень окисления −½ и входит в надпероксидный ион O₂⁻. Представители: KO₂, NaO₂, RbO₂.

Пероксид водорода H₂O₂

Физические свойства: бесцветная жидкость (в чистом виде), ρ = 1,44 г/см³, Tпл = −0,43°C, Tкип = 150,2°C (со взрывным разложением). Смешивается с водой в любых соотношениях. В быту используется 3% и 30% («пергидроль») растворы. Чистый H₂O₂ термически нестабилен и взрывоопасен. Слабая кислота: Ka = 2,4·10⁻¹².

Химические свойства:

H₂O₂ — универсальный окислитель и восстановитель одновременно (реакции диспропорционирования): в одних реакциях кислород повышает СО с −1 до 0 (H₂O₂ — восстановитель), в других понижает с −1 до −2 (H₂O₂ — окислитель).

Разложение (диспропорционирование):

2H₂O₂ → 2H₂O + O₂↑ (катализаторы: MnO₂, Fe²⁺, каталаза; свет, нагревание)

Как окислитель (СО(O): −1 → −2):

H₂O₂ + 2KI → I₂ + 2KOH (окисление йодид-иона)

H₂O₂ + 2FeSO₄ + H₂SO₄ → Fe₂(SO₄)₃ + 2H₂O (реакция Фентона)

H₂O₂ + Na₂SO₃ → Na₂SO₄ + H₂O

5H₂O₂ + 2KMnO₄ + 3H₂SO₄ → 2MnSO₄ + K₂SO₄ + 8H₂O + 5O₂↑ (H₂O₂ — восстановитель!)

Как восстановитель (СО(O): −1 → 0) — в реакции с сильными окислителями:

5H₂O₂ + 2KMnO₄ + 3H₂SO₄ → 2MnSO₄ + K₂SO₄ + 8H₂O + 5O₂↑

H₂O₂ + Cl₂ → 2HCl + O₂↑ (в присутствии сильного окислителя H₂O₂ — восстановитель)

Кислотные свойства:

H₂O₂ + 2NaOH → Na₂O₂ + 2H₂O (образование пероксида натрия)

H₂O₂ + Ba(OH)₂ → BaO₂↓ + 2H₂O

Получение:

Na₂O₂ + H₂SO₄(разб.) → Na₂SO₄ + H₂O₂ (лабораторный способ)

BaO₂ + H₂SO₄(разб.) → BaSO₄↓ + H₂O₂ (промышленный метод до XX века)

2-этилантрахинол + O₂ → 2-этилантрахинон + H₂O₂ (антрахиноновый метод — современный промышленный способ)

Применение: отбеливание тканей, целлюлозы и волос (окисление красящих пигментов); антисептик (3% раствор в медицине); производство эпоксидных смол и капролактама; ракетное топливо (концентрированный H₂O₂ — окислитель); очистка сточных вод; производство Na₂O₂ и других пероксидов.

Пероксид натрия Na₂O₂

Физические свойства: светло-жёлтый порошок, Tпл = 460°C, ρ = 2,81 г/см³. Сильный окислитель. Гигроскопичен — интенсивно поглощает влагу и CO₂ из воздуха, что используется в регенераторах кислорода.

Химические свойства:

Na₂O₂ — уникальный реагент: при взаимодействии с водой и CO₂ одновременно выделяет кислород и поглощает углекислый газ — что делает его незаменимым в изолирующих противогазах и на подводных лодках.

2Na₂O₂ + 2H₂O → 4NaOH + O₂↑ (диспропорционирование — и окислитель, и восстановитель)

2Na₂O₂ + 2CO₂ → 2Na₂CO₃ + O₂↑ (регенерация кислорода)

Na₂O₂ + H₂SO₄(разб.) → Na₂SO₄ + H₂O₂

Na₂O₂ + SO₂ → Na₂SO₄ (окисление SO₂)

2Na₂O₂ + 2H₂SO₄(конц.) → 2Na₂SO₄ + 2H₂O + O₂↑

Na₂O₂ + органическое вещество → воспламенение (сильный окислитель, требует осторожности)

Реакция Na₂O₂ с водой — диспропорционирование: кислород в O₂²⁻ (СО = −1) одна часть повышает до 0 (O₂), другая понижает до −2 (OH⁻). Это классическая задача ОГЭ/ЕГЭ на определение окислителя и восстановителя в одном соединении.

Получение:

2Na + O₂ → Na₂O₂ (сжигание натрия в избытке кислорода)

Применение: регенераторы кислорода в изолирующих противогазах, на подводных лодках и в космических аппаратах (поглощает CO₂ и выделяет O₂); отбеливатель в текстильной и бумажной промышленности; сильный окислитель в химическом синтезе.

Пероксид бария BaO₂

Физические свойства: белый порошок, Tпл = 450°C (с разложением), ρ = 4,96 г/см³. Малорастворим в воде. Более стабилен, чем Na₂O₂.

Химические свойства:

2BaO₂ → 2BaO + O₂↑ (разложение при нагревании выше 840°C)

BaO₂ + H₂SO₄(разб.) → BaSO₄↓ + H₂O₂ (получение H₂O₂)

BaO₂ + 2HCl → BaCl₂ + H₂O₂

2BaO₂ + 2H₂O → 2Ba(OH)₂ + O₂↑

Получение:

2BaO + O₂ → 2BaO₂ (окисление оксида бария при 500°C)

Ba(OH)₂ + H₂O₂ → BaO₂↓ + 2H₂O

Применение: исторически — получение H₂O₂; инициатор полимеризации; компонент некоторых термитных смесей.

Надпероксид калия KO₂

Физические свойства: оранжево-жёлтый порошок, Tпл = 380°C (с разложением), ρ = 2,14 г/см³. СО(O) = −½ (надпероксидный ион O₂⁻). Сильный окислитель.

Химические свойства:

4KO₂ + 2H₂O → 4KOH + 3O₂↑

4KO₂ + 2CO₂ → 2K₂CO₃ + 3O₂↑ (регенерация кислорода, выход O₂ выше, чем у Na₂O₂)

2KO₂ + H₂SO₄ → K₂SO₄ + H₂O₂ + O₂↑

KO₂ эффективнее Na₂O₂ как регенератор кислорода: из одного моля KO₂ при реакции с CO₂ выделяется ¾ моля O₂, тогда как из Na₂O₂ — ½ моля. Поэтому KO₂ применяется в современных изолирующих противогазах и аварийных системах жизнеобеспечения.

Получение:

K + O₂ → KO₂ (сжигание калия в избытке кислорода при комнатной температуре)

Применение: изолирующие противогазы и аварийные системы жизнеобеспечения (поглощает CO₂ и выделяет O₂); кислородно-дыхательные аппараты для горноспасателей и подводников.
