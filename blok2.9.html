Соли

1. Определение и классификация

Соли — ионные соединения, образованные катионами металлов (или NH₄⁺) и анионами кислотных остатков. С позиций теории электролитической диссоциации соли — электролиты, полностью или частично диссоциирующие в растворе на катион металла и анион кислотного остатка: MeₙAₘ → nMeᵐ⁺ + mAⁿ⁻.

По составу соли делятся на несколько групп. Средние (нормальные) соли — все атомы водорода кислоты замещены на металл: Na₂SO₄, CaCO₃, AlCl₃, K₃PO₄. Кислые соли — атомы водорода замещены не полностью, в составе есть кислотный водород: NaHSO₄, NaHCO₃, Ca(H₂PO₄)₂, KHSO₃. Образуются только многоосновными кислотами. Основные соли — гидроксогруппы основания замещены не полностью, в составе есть OH: Cu₂(OH)₂CO₃ (малахит), Mg(OH)Cl, Al(OH)₂Cl. Образуются только многокислотными основаниями. Двойные соли — содержат два разных катиона: KAl(SO₄)₂·12H₂O (квасцы), CaMg(CO₃)₂ (доломит). Смешанные соли — содержат два разных аниона: Ca(OCl)Cl (хлорная известь). Комплекснын соли Na[Al(OH)₄] (тетрагидроксоалюминат натрия)

Кислые соли образуются при недостатке основания или избытке кислоты: NaOH + H₂SO₄(изб.) → NaHSO₄ + H₂O. Основные соли — при недостатке кислоты или избытке основания: Cu(OH)₂ + CuSO₄ → Cu₂(OH)₂SO₄↓. В водных растворах кислые соли диссоциируют ступенчато: NaHCO₃ → Na⁺ + HCO₃⁻; HCO₃⁻ ⇌ H⁺ + CO₃²⁻ (слабая вторая ступень).

2. Физические свойства солей

2.1. Общие физические свойства

Большинство солей — твёрдые кристаллические вещества с ионной кристаллической решёткой. Температуры плавления, как правило, высокие: NaCl — 801°C, CaCO₃ — 825°C (разл.), CaSO₄ — 1460°C. В расплавленном состоянии и в водных растворах проводят электрический ток — являются электролитами. Исключения — соли со значительным ковалентным характером связи (HgCl₂, HgI₂, PbI₂) плавятся при значительно более низких температурах и плохо проводят ток.

Окраска солей определяется природой катиона и аниона. Катионы, окрашивающие соли: Cu²⁺ — синий или голубой (CuSO₄·5H₂O — синий купорос); Fe²⁺ — зеленоватый (FeSO₄·7H₂O — зелёный купорос); Fe³⁺ — жёлто-коричневый; Ni²⁺ — зелёный; Co²⁺ — розовый; Cr³⁺ — тёмно-зелёный или фиолетовый; MnO₄⁻ — фиолетовый; CrO₄²⁻ — жёлтый; Cr₂O₇²⁻ — оранжевый. Большинство солей щелочных металлов, Ca²⁺, Mg²⁺, Al³⁺, Zn²⁺ — белые или бесцветные.

Растворимость солей в воде разнообразна. Практически нерастворимы: BaSO₄, PbSO₄, CaSO₄ (малорастворим); AgCl, AgBr, AgI, PbCl₂ (малорастворим); BaCO₃, CaCO₃, SrCO₃, PbCO₃; Ca₃(PO₄)₂, Ba₃(PO₄)₂; Ag₃PO₄; CaF₂, BaF₂; FeS, CuS, PbS, HgS, Ag₂S. Хорошо растворимы все нитраты, все ацетаты, все соли натрия, калия и аммония, большинство хлоридов и сульфатов.

3. Химические свойства солей

3.1. Реакция с кислотами

Более сильная кислота вытесняет более слабую из её соли (реакция идёт если образуется осадок, газ или слабый электролит):

Na₂CO₃ + 2HCl → 2NaCl + H₂O + CO₂↑

Na₂S + 2HCl → 2NaCl + H₂S↑

Na₂SiO₃ + 2HCl → H₂SiO₃↓ + 2NaCl

CaCO₃ + H₂SO₄(разб.) → CaSO₄↓ + H₂O + CO₂↑ (реакция замедляется — CaSO₄ покрывает поверхность)

BaCl₂ + H₂SO₄ → BaSO₄↓ + 2HCl (нерастворимый осадок — качественная реакция на SO₄²⁻)

AgNO₃ + HCl → AgCl↓ + HNO₃ (качественная реакция на Cl⁻)

3FeS + 4HNO₃(разб.) → 3FeSO₄... нет; правильно: FeS + 2HCl → FeCl₂ + H₂S↑

Нерастворимые соли слабых кислот растворяются в сильных кислотах:

CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑

Cu₂S + 4H₂SO₄(конц., гор.) → 2CuSO₄ + S↓ + 4H₂O + SO₂↑... (нерастворимые сульфиды — только с окисляющими кислотами)

CuS + 8HNO₃(конц.) → CuSO₄ + 8NO₂↑ + 4H₂O

3.2. Реакция с основаниями

Щёлочи реагируют с солями если образуется осадок, газ или слабый электролит:

CuSO₄ + 2NaOH → Cu(OH)₂↓ + Na₂SO₄

FeCl₃ + 3NaOH → Fe(OH)₃↓ + 3NaCl

AlCl₃ + 3NaOH(недост.) → Al(OH)₃↓ + 3NaCl

AlCl₃ + 4NaOH(изб.) → Na[Al(OH)₄] + 3NaCl (амфотерный гидроксид растворяется в избытке щёлочи)

NH₄Cl + NaOH → NaCl + NH₃↑ + H₂O (качественная реакция на NH₄⁺)

(NH₄)₂SO₄ + Ca(OH)₂ → CaSO₄ + 2NH₃↑ + 2H₂O

MgCl₂ + 2NaOH → Mg(OH)₂↓ + 2NaCl

ZnSO₄ + 2NaOH(недост.) → Zn(OH)₂↓ + Na₂SO₄

ZnSO₄ + 4NaOH(изб.) → Na₂[Zn(OH)₄] + Na₂SO₄

3.3. Реакция солей между собой

Реакция идёт в растворе если образуется нерастворимый осадок:

BaCl₂ + Na₂SO₄ → BaSO₄↓ + 2NaCl

AgNO₃ + NaCl → AgCl↓ + NaNO₃

Pb(NO₃)₂ + 2KI → PbI₂↓(жёлтый) + 2KNO₃

CaCl₂ + Na₂CO₃ → CaCO₃↓ + 2NaCl

3BaCl₂ + 2K₃PO₄ → Ba₃(PO₄)₂↓ + 6KCl

Совместный (взаимный) гидролиз — реакция двух солей, каждая из которых гидролизуется по разному иону, идёт необратимо:

AlCl₃ + 3Na₂CO₃ + 3H₂O → 2Al(OH)₃↓ + 3CO₂↑ + 6NaCl

Cr₂(SO₄)₃ + 3Na₂CO₃ + 3H₂O → 2Cr(OH)₃↓ + 3CO₂↑ + 3Na₂SO₄

Совместный гидролиз — важная тема ЕГЭ. Al₂S₃, Al₂(CO₃)₃, Cr₂(CO₃)₃ в водном растворе не существуют — они немедленно и необратимо гидролизуются: Al₂S₃ + 6H₂O → 2Al(OH)₃ + 3H₂S↑. Поэтому эти соли нельзя получить в водном растворе — только сухим синтезом.

3.4. Реакция с металлами

Более активный металл вытесняет менее активный из раствора его соли (ряд активности). Щелочные и щелочноземельные металлы реагируют сначала с водой, а не с солью металла:

Fe + CuSO₄ → FeSO₄ + Cu↓ (железо вытесняет медь)

Cu + 2AgNO₃ → Cu(NO₃)₂ + 2Ag↓ (медь вытесняет серебро)

Zn + FeSO₄ → ZnSO₄ + Fe↓ (цинк вытесняет железо)

Zn + CuSO₄ → ZnSO₄ + Cu↓

2Fe + 3CuSO₄ → Fe₂(SO₄)₃ + 3Cu↓... нет: Fe + CuSO₄ → FeSO₄ + Cu (Fe → Fe²⁺, не Fe³⁺ — Cu²⁺ недостаточно сильный окислитель)

Взаимодействие Fe с избытком соли Fe³⁺:

Fe + 2FeCl₃ → 3FeCl₂ (Fe восстанавливает Fe³⁺ до Fe²⁺)

3.5. Гидролиз солей

Гидролиз — взаимодействие иона соли с водой, приводящее к смещению равновесия диссоциации воды и изменению pH раствора. Характер гидролиза определяется природой кислоты и основания, образовавших соль.

Соль сильной кислоты и сильного основания — гидролизу не подвергается, раствор нейтральный (pH = 7): NaCl, K₂SO₄, Ba(NO₃)₂.

Соль сильной кислоты и слабого основания — гидролиз по катиону, раствор кислый (pH < 7):

Al³⁺ + H₂O ⇌ Al(OH)²⁺ + H⁺ (первая ступень)

NH₄⁺ + H₂O ⇌ NH₃·H₂O + H⁺

Cu²⁺ + 2H₂O ⇌ Cu(OH)₂ + 2H⁺ (упрощённо)

Соль слабой кислоты и сильного основания — гидролиз по аниону, раствор щелочной (pH > 7):

CO₃²⁻ + H₂O ⇌ HCO₃⁻ + OH⁻

S²⁻ + H₂O ⇌ HS⁻ + OH⁻

CH₃COO⁻ + H₂O ⇌ CH₃COOH + OH⁻

SiO₃²⁻ + H₂O ⇌ HSiO₃⁻ + OH⁻

Соль слабой кислоты и слабого основания — гидролиз по катиону и аниону одновременно, pH близок к 7 (зависит от констант):

CH₃COONH₄ + H₂O ⇌ CH₃COOH + NH₃·H₂O (pH ≈ 7)

Al₂S₃ + 6H₂O → 2Al(OH)₃↓ + 3H₂S↑ (необратимый полный гидролиз)

Степень гидролиза возрастает при разбавлении раствора и при нагревании. Для подавления гидролиза соли, образованной слабым основанием, в раствор добавляют соответствующую кислоту (например, FeCl₃ хранят в слегка подкисленном растворе). Для усиления гидролиза — разбавляют и нагревают.

3.6. Термическое разложение солей

Карбонаты разлагаются при нагревании (активность металла определяет температуру разложения):

CaCO₃ → CaO + CO₂↑ (T > 900°C)

MgCO₃ → MgO + CO₂↑ (T > 540°C)

ZnCO₃ → ZnO + CO₂↑ (T > 300°C)

CuCO₃ → CuO + CO₂↑ (T ≈ 200°C)

Na₂CO₃, K₂CO₃ — не разлагаются до температур плавления (щелочные металлы — устойчивые карбонаты)

Нитраты разлагаются в зависимости от положения металла в ряду активности:

2NaNO₃ → 2NaNO₂ + O₂↑ (T ≈ 380°C — щелочные металлы → нитриты)

2Ca(NO₃)₂ → 2CaO + 4NO₂↑ + O₂↑ (щелочноземельные → оксид)

2Cu(NO₃)₂ → 2CuO + 4NO₂↑ + O₂↑ (металлы средней активности → оксид)

2AgNO₃ → 2Ag + 2NO₂↑ + O₂↑ (малоактивные металлы → металл)

2Fe(NO₃)₃ → Fe₂O₃ + 6NO₂↑ + ... (точнее: Fe(NO₃)₃ при нагревании → Fe₂O₃)

NH₄NO₃ → N₂O↑ + 2H₂O (T = 250–270°C)

NH₄NO₃ → N₂↑ + 2H₂O + ½O₂ (при взрыве)

Сульфаты разлагаются при очень высоких температурах:

CaSO₄ → CaO + SO₃↑ (T > 1400°C)

2FeSO₄ → Fe₂O₃ + SO₂↑ + SO₃↑ (T > 480°C)

CuSO₄ → CuO + SO₃↑ (T > 700°C)

Соли аммония разлагаются при нагревании (летучее основание):

NH₄Cl → NH₃↑ + HCl↑ (возгонка при T > 337°C)

(NH₄)₂CO₃ → 2NH₃↑ + H₂O + CO₂↑ (T > 60°C)

(NH₄)₂SO₄ → NH₃↑ + NH₄HSO₄ (T = 235°C); далее при T > 400°C → N₂ + SO₂ + H₂O

NH₄HCO₃ → NH₃↑ + H₂O + CO₂↑ (T > 36°C — очень легко разлагается)

3.7. Окислительно-восстановительные реакции солей

Соли, содержащие катион металла с переменной валентностью, могут выступать окислителями или восстановителями:

2FeCl₃ + Cu → 2FeCl₂ + CuCl₂ (Fe³⁺ — окислитель)

2FeCl₃ + Fe → 3FeCl₂ (Fe³⁺ — окислитель, Fe⁰ — восстановитель)

2FeCl₃ + H₂S → 2FeCl₂ + S↓ + 2HCl

2KMnO₄ + 5Na₂SO₃ + 3H₂SO₄ → 2MnSO₄ + 5Na₂SO₄ + K₂SO₄ + 3H₂O

K₂Cr₂O₇ + 6FeSO₄ + 7H₂SO₄ → Cr₂(SO₄)₃ + 3Fe₂(SO₄)₃ + K₂SO₄ + 7H₂O

Соли-восстановители (содержат анион с пониженной степенью окисления):

Na₂SO₃ + Cl₂ + H₂O → Na₂SO₄ + 2HCl

2KI + Cl₂ → 2KCl + I₂ (I⁻ — восстановитель)

Na₂S + I₂ → 2NaI + S↓ (S²⁻ — восстановитель)

4. Получение солей

4.1. Реакция металла с неметаллом (прямой синтез)

2Na + Cl₂ → 2NaCl

Fe + S → FeS (при нагревании)

Cu + Cl₂ → CuCl₂

4.2. Реакция металла с кислотой

Zn + 2HCl → ZnCl₂ + H₂↑

3Cu + 8HNO₃(разб.) → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O

Fe + 2H₂SO₄(разб.) → FeSO₄ + H₂↑

Cu + 2H₂SO₄(конц.) → CuSO₄ + SO₂↑ + 2H₂O

4.3. Реакция металла с раствором соли

Fe + CuSO₄ → FeSO₄ + Cu

Cu + 2AgNO₃ → Cu(NO₃)₂ + 2Ag

4.4. Реакция основного оксида с кислотным оксидом

CaO + CO₂ → CaCO₃

CaO + SiO₂ → CaSiO₃ (при сплавлении)

Na₂O + SO₃ → Na₂SO₄

3CaO + P₂O₅ → Ca₃(PO₄)₂

4.5. Реакция основания с кислотой (нейтрализация)

NaOH + HCl → NaCl + H₂O

Ca(OH)₂ + H₂SO₄ → CaSO₄↓ + 2H₂O

Cu(OH)₂ + 2HNO₃ → Cu(NO₃)₂ + 2H₂O

3NaOH + H₃PO₄ → Na₃PO₄ + 3H₂O

4.6. Реакция основания с кислотным оксидом

2NaOH + CO₂ → Na₂CO₃ + H₂O

Ca(OH)₂ + SO₂ → CaSO₃↓ + H₂O

2NaOH + SiO₂ → Na₂SiO₃ + H₂O (при сплавлении)

4.7. Реакция кислоты с основным оксидом

CuO + H₂SO₄ → CuSO₄ + H₂O

Fe₂O₃ + 6HCl → 2FeCl₃ + 3H₂O

Al₂O₃ + 3H₂SO₄ → Al₂(SO₄)₃ + 3H₂O

4.8. Реакция кислоты с солью

CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑

Na₂CO₃ + H₂SO₄ → Na₂SO₄ + H₂O + CO₂↑

Na₂SiO₃ + 2HCl → H₂SiO₃↓ + 2NaCl

4.9. Реакция основания с солью

CuSO₄ + 2NaOH → Cu(OH)₂↓ + Na₂SO₄

Na₂CO₃ + Ca(OH)₂ → CaCO₃↓ + 2NaOH

(NH₄)₂SO₄ + 2NaOH → Na₂SO₄ + 2NH₃↑ + 2H₂O

4.10. Реакция двух солей (только при выпадении осадка)

BaCl₂ + Na₂SO₄ → BaSO₄↓ + 2NaCl

AgNO₃ + NaCl → AgCl↓ + NaNO₃

Pb(NO₃)₂ + 2KI → PbI₂↓ + 2KNO₃

4.11. Реакция амфотерного оксида или гидроксида со щёлочью

Al₂O₃ + 2NaOH → 2NaAlO₂ + H₂O (при сплавлении)

Zn(OH)₂ + 2KOH → K₂[Zn(OH)₄]

Al(OH)₃ + NaOH → Na[Al(OH)₄]

5. Физические свойства и применение важнейших солей

5.1. Хлорид натрия NaCl

Бесцветные кристаллы, Tпл = 801°C, ρ = 2,17 г/см³. Растворимость 35,9 г/100 мл (мало меняется с температурой). Природный минерал — галит (каменная соль), сильвинит (KCl·NaCl), галургит. Гидролизу не подвергается, раствор нейтральный.

Применение: пищевая промышленность (консервант, вкусовая добавка); электролиз раствора → NaOH, Cl₂, H₂; электролиз расплава → Na и Cl₂; производство Na₂CO₃ (аммиачный метод Сольвея); дорожная служба (антиобледенитель); производство HCl; медицина (физиологический раствор 0,9%).

5.2. Карбонат натрия Na₂CO₃ (кальцинированная сода)

Белый порошок, Tпл = 851°C, хорошо растворим (21,5 г/100 мл при 20°C). Гидролиз по аниону: CO₃²⁻ + H₂O ⇌ HCO₃⁻ + OH⁻; раствор сильно щелочной (pH ≈ 11,6). Образует кристаллогидраты: Na₂CO₃·H₂O, Na₂CO₃·7H₂O, Na₂CO₃·10H₂O (стиральная сода). Реагирует с CO₂: Na₂CO₃ + CO₂ + H₂O → 2NaHCO₃.

Применение: производство стекла (Na₂CO₃ + SiO₂ → Na₂SiO₃ + CO₂ — при варке стекла); производство мыла (омыление жиров в щелочной среде); целлюлозно-бумажная промышленность; синтетические моющие средства (умягчитель воды); производство NaOH (каустификация); пищевая промышленность (разрыхлитель в составе пекарского порошка).

5.3. Гидрокарбонат натрия NaHCO₃ (пищевая сода)

Белый порошок, умеренно растворим (9,6 г/100 мл при 20°C). Раствор слабощелочной (pH ≈ 8,3). При нагревании разлагается:

2NaHCO₃ → Na₂CO₃ + H₂O + CO₂↑ (T > 60°C)

Реагирует и с кислотами, и со щелочами:

NaHCO₃ + HCl → NaCl + H₂O + CO₂↑

NaHCO₃ + NaOH → Na₂CO₃ + H₂O

Применение: кулинария (разрыхлитель теста — при нагревании выделяет CO₂); медицина (антацид — нейтрализация избыточной HCl в желудке; средство при ожогах кислотами); огнетушители (порошковые и углекислотные — реакция NaHCO₃ с кислотой); лёгкая промышленность (обработка кожи).

5.4. Сульфат кальция CaSO₄

Белое вещество, малорастворимо (0,21 г/100 мл при 20°C). Существует в нескольких гидратных формах: CaSO₄·2H₂O — гипс (природный минерал); CaSO₄·0,5H₂O — алебастр (жжёный гипс, полугидрат, получают нагреванием гипса при 120–180°C); CaSO₄ — ангидрит (безводный, Tпл = 1460°C).

CaSO₄·2H₂O → CaSO₄·0,5H₂O + 1,5H₂O (T = 120–180°C — получение алебастра)

CaSO₄·0,5H₂O + 1,5H₂O → CaSO₄·2H₂O (схватывание — гипсование)

Применение: строительство (алебастр — гипсовые штукатурки, перегородки, лепнина); медицина (гипсовые повязки при переломах); производство цемента (CaSO₄ добавляют для регулирования сроков схватывания); производство серной кислоты (CaSO₄ — отход экстракционного метода получения H₃PO₄).

5.5. Сульфат бария BaSO₄

Белый порошок, практически нерастворим в воде (0,00024 г/100 мл) и в кислотах. Не токсичен несмотря на то что Ba²⁺ в растворимых соединениях ядовит. Природный минерал — барит.

Применение: рентгеноконтрастное вещество («бариевая каша» — суспензия BaSO₄, используется при рентгеноскопии ЖКТ, так как поглощает рентгеновские лучи и при этом нерастворим и нетоксичен); белый пигмент (бланфикс); наполнитель резины, пластмасс, бумаги; буровые растворы (барит утяжеляет буровой раствор при нефтедобыче).

5.6. Нитрат аммония NH₄NO₃ (аммиачная селитра)

Белые гигроскопичные кристаллы, Tпл = 169°C, хорошо растворим (190 г/100 мл при 20°C). Разлагается при нагревании — тип разложения зависит от температуры:

NH₄NO₃ → N₂O↑ + 2H₂O (T = 250–270°C — умеренный нагрев)

2NH₄NO₃ → 2N₂↑ + O₂↑ + 4H₂O (T > 300°C — опасный разогрев)

NH₄NO₃ → N₂↑ + 2H₂O + ½O₂ (взрыв)

Применение: азотное удобрение (крупнейший по объёму производства; содержит 34% N); взрывчатые вещества (аммонал, аммонит — смеси с горючими веществами; аммиачная селитра — окислитель).

5.7. Нитрат калия KNO₃ (калийная селитра)

Бесцветные кристаллы, Tпл = 334°C, хорошо растворим. При нагревании даёт нитрит: 2KNO₃ → 2KNO₂ + O₂ (T = 400°C). Сильный окислитель в расплаве.

Применение: азотно-калийное удобрение; компонент чёрного пороха (KNO₃ + C + S → смесь, инициирующая воспламенение); консервант в пищевой промышленности (E252 — посолочная смесь для мяса).

5.8. Карбонат кальция CaCO₃

Белое вещество, практически нерастворимо в воде (0,00013 г/100 мл), но растворяется в воде, насыщенной CO₂: CaCO₃ + CO₂ + H₂O → Ca(HCO₃)₂ (образование жёсткости воды; при нагревании реакция обратима — выпадение накипи). Природные разновидности: кальцит, мрамор, известняк, мел, ракушечник.

CaCO₃ → CaO + CO₂↑ (T > 900°C — обжиг известняка)

CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑ (качественная реакция на CO₃²⁻)

Применение: производство CaO (негашёной извести) и цемента; производство стекла (CaCO₃ → CaO при варке); металлургия (флюс для удаления SiO₂); наполнитель резины, красок, пластмасс; мел — пигмент, пищевая добавка (E170); мрамор — строительный и декоративный материал.

5.9. Сульфат меди(II) CuSO₄·5H₂O (медный купорос)

Синие кристаллы (цвет обусловлен аквакомплексом [Cu(H₂O)₄]²⁺), Tпл = 110°C (потеря кристаллизационной воды). Безводный CuSO₄ — белый порошок; поглощение воды сопровождается посинением — аналитическая реакция на воду. Хорошо растворим (20 г/100 мл при 20°C).

CuSO₄·5H₂O → CuSO₄·3H₂O → CuSO₄·H₂O → CuSO₄ (ступенчатая дегидратация)

CuSO₄ + 5H₂O → CuSO₄·5H₂O (гидратация — посинение)

Применение: сельское хозяйство (фунгицид — «бордоская жидкость» CuSO₄ + Ca(OH)₂; протравливание семян); гальваника (электролит для меднения); аналитическая химия (реагент на воду, качественные реакции); производство других соединений меди; антисептик для пропитки древесины.

5.10. Силикат натрия Na₂SiO₃ (жидкое стекло)

Белое твёрдое вещество или вязкий водный раствор (жидкое стекло). Tпл = 1088°C. Хорошо растворим в горячей воде. Раствор сильно щелочной (сильный гидролиз по SiO₃²⁻). При действии кислот или CO₂ выпадает белый студенистый осадок H₂SiO₃.

Применение: огнезащитные пропитки (ткани, дерево, бумага — после обработки Na₂SiO₃ покрываются тонким слоём H₂SiO₃ при контакте с CO₂ воздуха); канцелярский силикатный клей; производство силикагеля; пропитка яиц (консервирование — поры скорлупы закупориваются); строительство (жидкое стекло в цементных составах).

5.11. Перманганат калия KMnO₄

Тёмно-фиолетовые кристаллы с металлическим блеском, Tпл = 240°C (с разложением). Раствор малиново-фиолетового цвета. Сильный окислитель. Разлагается при нагревании и на свету. Подробные химические свойства — в разделе «Кислоты» (HMnO₄).

Применение: медицина (антисептик, дезинфектант — 0,01–0,1% раствор); аналитическая химия (перманганатометрия — объёмный метод анализа); органический синтез (мягкое и жёсткое окисление органических веществ); лабораторное получение O₂ (разложение при нагревании); обеззараживание воды.

5.12. Дихромат калия K₂Cr₂O₇ (бихромат, «хромпик»)

Оранжево-красные кристаллы, Tпл = 398°C. Сильный окислитель в кислой среде (Cr⁶⁺ → Cr³⁺). Токсичен — соединения Cr⁶⁺ канцерогенны. В щелочной среде переходит в жёлтый хромат K₂CrO₄.

Применение: кожевенная промышленность (хромовое дубление кожи); текстиль (протрава при крашении); аналитическая химия (хроматометрия, «хромовая смесь» K₂Cr₂O₇ + конц. H₂SO₄ — для мытья химической посуды от органических загрязнений); пиротехника (окислитель).
