Бинарные соединения

Бинарные соединения — вещества, молекулы или формульные единицы которых образованы атомами ровно двух химических элементов. Общая формула записывается как EₓYᵧ, где E и Y — символы элементов, x и y — стехиометрические коэффициенты. К бинарным соединениям относятся оксиды, гидриды, галогениды, сульфиды, нитриды, карбиды, фосфиды, силициды и ряд других классов. Их изучение составляет основу неорганической химии, поскольку именно бинарные соединения являются исходными продуктами для синтеза более сложных веществ.

Сульфиды

1. Общая характеристика и классификация

Сульфиды — бинарные соединения элементов с серой, в которых сера имеет степень окисления −2. Общая формула: MeₓSᵧ. Сульфиды широко распространены в природе — многие из них являются важнейшими рудами: пирит FeS₂ (серный колчедан), галенит PbS, сфалерит ZnS, халькопирит CuFeS₂, киноварь HgS, аргентит Ag₂S.

По растворимости сульфиды делятся на три группы. Растворимые — сульфиды щелочных металлов и аммония: Na₂S, K₂S, (NH₄)₂S. Малорастворимые или нерастворимые в воде, но растворимые в кислотах — сульфиды металлов средней активности: MnS, FeS, CoS, NiS, ZnS. Практически нерастворимые ни в воде, ни в разбавленных кислотах — сульфиды малоактивных металлов: CuS, Ag₂S, PbS, HgS, Bi₂S₃. Эта классификация прямо связана с произведениями растворимости (ПР): ПР(HgS) = 4·10⁻⁵³ — рекордно малое значение.

2. Химические свойства сульфидов

Гидролиз растворимых сульфидов — сульфид-ион S²⁻ является сопряжённым основанием очень слабой кислоты H₂S (Ka₂ = 1·10⁻¹⁴), поэтому растворы сульфидов сильно гидролизованы и имеют резко щелочную реакцию:

S²⁻ + H₂O ⇌ HS⁻ + OH⁻ (первая ступень гидролиза)

Na₂S + 2H₂O ⇌ 2NaOH + H₂S↑ (при нагревании — полный гидролиз)

Реакция с кислотами — сильные кислоты вытесняют H₂S из растворимых сульфидов:

Na₂S + 2HCl → 2NaCl + H₂S↑

FeS + 2HCl → FeCl₂ + H₂S↑ (лабораторный способ получения H₂S)

ZnS + H₂SO₄(разб.) → ZnSO₄ + H₂S↑

Нерастворимые сульфиды (CuS, HgS, PbS) в разбавленных кислотах не растворяются — только в концентрированных окисляющих кислотах:

3CuS + 8HNO₃(разб.) → 3CuSO₄ + 8NO↑ + 4H₂O (нет — неверно; правильно:)

CuS + 8HNO₃(конц.) → CuSO₄ + 8NO₂↑ + 4H₂O

HgS + 2HCl + 2HNO₃(конц.) → HgCl₂ + S↓ + 2NO↑ + 2H₂O (царская водка растворяет HgS)

Реакция с кислородом (обжиг) — важнейшая реакция в металлургии:

2ZnS + 3O₂ → 2ZnO + 2SO₂↑ (обжиг сфалерита)

4FeS₂ + 11O₂ → 2Fe₂O₃ + 8SO₂↑ (обжиг пирита — источник SO₂ для производства H₂SO₄)

2PbS + 3O₂ → 2PbO + 2SO₂↑ (обжиг галенита)

Cu₂S + O₂ → 2Cu + SO₂↑ (высокотемпературный обжиг — получение Cu)

Реакция с щелочами — растворимые сульфиды со щелочами не реагируют. Нерастворимые сульфиды амфотерных металлов (SnS, As₂S₃, As₂S₅) растворяются в щелочах или сульфидах щелочных металлов:

As₂S₃ + 6NaOH → Na₃AsO₃ + Na₃AsS₃ + 3H₂O

SnS + Na₂S → Na₂SnS₂ (тиостаннат натрия — в растворе Na₂S)

Реакция с галогенами и другими окислителями:

PbS + 4Cl₂ + 4H₂O → PbCl₂ + H₂SO₄ + 6HCl

Na₂S + 4H₂O₂ → Na₂SO₄ + 4H₂O (окисление сульфид-иона пероксидом)

3. Получение и применение сульфидов

Получение сульфидов:

Fe + S → FeS (при нагревании — прямой синтез)

2Na + S → Na₂S (при нагревании)

Cu + S → CuS (при нагревании)

Zn + S → ZnS (при нагревании)

ZnSO₄ + Na₂S → ZnS↓ + Na₂SO₄ (осаждение из раствора)

CuSO₄ + H₂S → CuS↓ + H₂SO₄ (осаждение в кислой среде)

Na₂SO₄ + 4C → Na₂S + 4CO (восстановление сульфата углём, T > 800°C)

Применение: ZnS — люминофор в электронно-лучевых трубках и флуоресцентных покрытиях, пигмент (литопон ZnS·BaSO₄); PbS — полупроводниковый фотодетектор ИК-излучения; CdS — жёлтый пигмент, фотоэлектрический элемент; FeS₂ — производство H₂SO₄; Na₂S — производство красителей, дубление кожи, фотография; Ag₂S — фотография.

4. Качественная реакция на сульфид-ион и H₂S

Pb(NO₃)₂ + H₂S → PbS↓(чёрный) + 2HNO₃

Pb²⁺ + S²⁻ → PbS↓ (чёрный осадок — качественная реакция)

Почернение бумаги, смоченной раствором Pb(NO₃)₂ — классический способ обнаружения H₂S в атмосфере. Иногда этим объясняют потемнение старых картин, написанных белилами на основе PbSO₄ или Pb(OH)₂ — они медленно превращаются в чёрный PbS под воздействием атмосферного H₂S.
