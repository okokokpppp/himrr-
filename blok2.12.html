Элементы VA группы: азот и фосфор

N · P · As · Sb · Bi

1. Общая характеристика группы

Группа VA включает азот N (Z=7), фосфор P (Z=15), мышьяк As (Z=33), сурьму Sb (Z=51) и висмут Bi (Z=83). Конфигурация внешнего уровня: ns²np³. Три неспаренных p-электрона определяют характерную степень окисления −3. Высшая степень окисления: +5.

В группе наблюдается переход от неметалла к металлу: азот и фосфор — неметаллы; мышьяк и сурьма — металлоиды; висмут — металл. У висмута выражен эффект инертной пары: Bi(+3) значительно устойчивее Bi(+5). Кислотный характер высших оксидов убывает от N₂O₅ к Bi₂O₅.

Азот принципиально отличается от остальных элементов группы: у него нет d-орбиталей, поэтому максимальная ковалентность азота равна четырём (NH₄⁺). Фосфор и последующие элементы расширяют октет и образуют пятивалентные соединения (PCl₅, PF₅).

2. Азот (N, Z = 7)

2.1. Физические свойства

Азот N₂ — бесцветный газ без запаха и вкуса, Tкип = −195,8°C, Tпл = −210,0°C. M = 28 г/моль, плохо растворим в воде. Составляет 78,09% атмосферы. Жидкий азот — прозрачная жидкость, кипящая при −196°C. Молекула N₂ имеет тройную связь N≡N с энергией 945 кДж/моль — одна из самых прочных связей в химии, что и обусловливает инертность азота при обычных условиях.

2.2. Химические свойства азота N₂

Азот крайне инертен при обычных условиях и реагирует только при высоких температурах или в особых условиях.

N₂ + 3H₂ ⇌ 2NH₃ (Fe-катализатор, T = 450°C, P = 150–300 атм — синтез Габера–Боша)

N₂ + O₂ ⇌ 2NO (T > 2000°C — электрическая дуга или молния, ΔH = +180,5 кДж/моль)

6Li + N₂ → 2Li₃N (единственный щелочной металл, реагирующий с N₂ при н.у.)

3Ca + N₂ → Ca₃N₂ (при нагревании)

2Al + N₂ → 2AlN (T > 800°C)

2.3. Аммиак NH₃

Физические свойства: бесцветный газ с резким запахом, Tкип = −33,4°C. Легче воздуха (M = 17 г/моль). При 20°C растворяется около 700 объёмов на 1 объём воды. Молекула пирамидальная (sp³), образует водородные связи.

Химические свойства:

Основные свойства:

NH₃ + H₂O ⇌ NH₄⁺ + OH⁻ (Kb = 1,8·10⁻⁵ — слабое основание)

NH₃ + HCl → NH₄Cl (белый дым — качественная реакция на NH₃ и HCl)

2NH₃ + H₂SO₄ → (NH₄)₂SO₄

NH₃ + HNO₃ → NH₄NO₃

Восстановительные свойства:

4NH₃ + 3O₂ → 2N₂ + 6H₂O (горение без катализатора)

4NH₃ + 5O₂ → 4NO + 6H₂O (Pt–Rh катализатор, T = 850°C)

2NH₃ + 3CuO → 3Cu + N₂ + 3H₂O

8NH₃ + 3Cl₂ → N₂ + 6NH₄Cl (NH₃ в избытке)

Реакции с металлами и комплексообразование:

2Na + 2NH₃(г) → 2NaNH₂ + H₂↑ (T = 300°C)

Cu(OH)₂ + 4NH₃ → [Cu(NH₃)₄](OH)₂ (ярко-синий — качественная реакция на Cu²⁺)

AgCl + 2NH₃ → [Ag(NH₃)₂]Cl (растворение осадка AgCl в аммиаке)

Получение:

N₂ + 3H₂ ⇌ 2NH₃ (процесс Габера–Боша — промышленный)

2NH₄Cl + Ca(OH)₂ → CaCl₂ + 2NH₃↑ + 2H₂O (нагревание — лабораторный)

Применение: производство азотных удобрений (NH₄NO₃, карбамид, (NH₄)₂SO₄); производство HNO₃; холодильный агент в промышленных установках; нашатырный спирт (10% раствор) в медицине; синтез полимеров (нейлон, капрон).

2.4. Оксиды азота

N₂O — бесцветный газ, СО(N) = +1, несолеобразующий. Поддерживает горение: 2N₂O → 2N₂ + O₂. Применяется в медицине как анестетик («веселящий газ») и как пропеллент в аэрозолях.

NO — бесцветный газ, СО(N) = +2, несолеобразующий, нерастворим в воде. Мгновенно окисляется на воздухе:

2NO + O₂ → 2NO₂ (бурый газ)

NO₂ — бурый газ с резким запахом, СО(N) = +4. При охлаждении: 2NO₂ ⇌ N₂O₄ (бесцветный). Реагирует с водой:

3NO₂ + H₂O → 2HNO₃ + NO↑ (ключевая реакция в производстве HNO₃)

2NO₂ + 2NaOH → NaNO₃ + NaNO₂ + H₂O

N₂O₅ — белое кристаллическое вещество, ангидрид HNO₃. Нестабилен: 2N₂O₅ → 4NO₂ + O₂; N₂O₅ + H₂O → 2HNO₃.

2.5. Азотная кислота HNO₃

Физические свойства: бесцветная жидкость (Tкип = 83°C), желтеет на свету из-за растворения NO₂. Смешивается с водой в любых соотношениях. Сильная одноосновная кислота.

Химические свойства:

HNO₃ — сильный окислитель; водород при реакции с металлами не выделяется никогда. Продукт восстановления зависит от концентрации и активности металла:

Cu + 4HNO₃(конц.) → Cu(NO₃)₂ + 2NO₂↑ + 2H₂O

3Cu + 8HNO₃(разб.) → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O

4Zn + 10HNO₃(оч.разб.) → 4Zn(NO₃)₂ + NH₄NO₃ + 3H₂O

Fe + 4HNO₃(разб.) → Fe(NO₃)₃ + NO↑ + 2H₂O

3Ag + 4HNO₃(разб.) → 3AgNO₃ + NO↑ + 2H₂O

Пассивация: Fe, Al, Cr не реагируют с конц. HNO₃ при н.у. — образуется защитная оксидная плёнка. Именно поэтому конц. HNO₃ перевозят в стальных и алюминиевых цистернах.

Реакции с неметаллами:

S + 6HNO₃(конц.) → H₂SO₄ + 6NO₂↑ + 2H₂O

C + 4HNO₃(конц.) → CO₂↑ + 4NO₂↑ + 2H₂O

P + 5HNO₃(конц.) → H₃PO₄ + 5NO₂↑ + H₂O

Разложение при нагревании и на свету:

4HNO₃ → 4NO₂↑ + O₂↑ + 2H₂O

Получение (метод Оствальда, 3 стадии):

4NH₃ + 5O₂ → 4NO + 6H₂O (Pt–Rh, T = 850°C)

2NO + O₂ → 2NO₂

3NO₂ + H₂O → 2HNO₃ + NO↑ (NO возвращают в цикл)

Применение: производство азотных удобрений (NH₄NO₃ — аммиачная селитра); взрывчатые вещества (тротил, нитроглицерин, нитроцеллюлоза); нитрование органических соединений в синтезе красителей и лекарств; аффинаж металлов; «царская водка» (HNO₃ + 3HCl) — растворитель Au и Pt.

2.6. Нитраты

Нитраты всех металлов растворимы в воде — исключений нет. Все нитраты являются окислителями при нагревании. Тип разложения зависит от положения металла в ряду активности:

2NaNO₃ → 2NaNO₂ + O₂↑ (щелочные металлы — до нитрита, T ≈ 380°C)

2Cu(NO₃)₂ → 2CuO + 4NO₂↑ + O₂↑ (металлы средней активности — до оксида)

2AgNO₃ → 2Ag + 2NO₂↑ + O₂↑ (малоактивные металлы — до металла)

NH₄NO₃ → N₂O↑ + 2H₂O (T = 250°C)

3. Фосфор (P, Z = 15)

3.1. Аллотропные модификации

Белый фосфор P₄ — молекулярная решётка из тетраэдрических молекул. Белое восковидное вещество, Tпл = 44°C. Светится в темноте. Крайне ядовит. Самовоспламеняется на воздухе при 40°C. Хранится под водой.

Красный фосфор — аморфный полимер. Тёмно-красный порошок, не светится, практически не токсичен, загорается при ~240°C, нерастворим в воде и органических растворителях. Получают нагреванием белого фосфора при 250–300°C без воздуха.

Чёрный фосфор — наиболее термодинамически устойчивая форма, слоистая структура (аналог графита), полупроводник. Получают при давлении ~12 000 атм.

Реакционная способность модификаций: белый >> красный >> чёрный. В практике используют преимущественно красный фосфор как безопасный.

3.2. Физические свойства

Белый фосфор — мягкое восковидное вещество, Tпл = 44°C, летуч, светится в темноте, ядовит. Красный — порошок тёмно-красного цвета, тугоплавкий, нелетуч. Чёрный — блестящие чешуйки, похожие на графит.

3.3. Химические свойства

Реакции с неметаллами:

P₄ + 5O₂ → 2P₂O₅ (горение, белый дым)

P₄ + 6Cl₂(недост.) → 4PCl₃

P₄ + 10Cl₂(избыт.) → 4PCl₅

Реакции с металлами:

2P + 3Ca → Ca₃P₂ (фосфид кальция)

Ca₃P₂ + 6H₂O → 3Ca(OH)₂ + 2PH₃↑ (фосфин — объяснение «болотных огней»)

Реакции со щёлочью (только белый фосфор, диспропорционирование):

P₄ + 3NaOH + 3H₂O → PH₃↑ + 3NaH₂PO₂

Реакции с концентрированными кислотами-окислителями:

P + 5HNO₃(конц.) → H₃PO₄ + 5NO₂↑ + H₂O

3.4. Важнейшие соединения фосфора

Оксид фосфора(V) P₂O₅ — белый летучий порошок, один из лучших осушителей. Реагирует с водой:

P₂O₅ + 3H₂O → 2H₃PO₄

P₂O₅ + H₂O → 2HPO₃ (метафосфорная кислота, при недостатке воды)

Ортофосфорная кислота H₃PO₄ — бесцветные кристаллы (Tпл = 42,4°C), в продаже — 85% раствор. Трёхосновная кислота средней силы. Не является окислителем. Образует три ряда солей:

NaOH + H₃PO₄ → NaH₂PO₄ + H₂O (дигидрофосфат)

2NaOH + H₃PO₄ → Na₂HPO₄ + 2H₂O (гидрофосфат)

3NaOH + H₃PO₄ → Na₃PO₄ + 3H₂O (фосфат)

Хлориды фосфора бурно гидролизуются:

PCl₃ + 3H₂O → H₃PO₃ + 3HCl

PCl₅ + 4H₂O → H₃PO₄ + 5HCl

3.5. Получение

Ca₃(PO₄)₂ + 3SiO₂ + 5C → 3CaSiO₃ + 5CO + 2P↑ (T = 1400–1500°C, электропечь)

Ca₃(PO₄)₂ + 3H₂SO₄ → 2H₃PO₄ + 3CaSO₄↓ (получение H₃PO₄ — экстракционный метод)

3.6. Применение

Красный фосфор — боковая поверхность спичечного коробка; пиротехника.

H₃PO₄ и фосфаты — производство фосфорных удобрений (суперфосфат, аммофос — ~90% мирового потребления фосфора); пищевая добавка E338 в газированных напитках; антикоррозионная обработка металлов (фосфатирование); зубные пасты.

P₂O₅ — осушитель газов и растворителей.

PCl₃ и PCl₅ — реагенты в органическом синтезе.

4. Мышьяк, сурьма, висмут (As, Sb, Bi)

4.1. Физические свойства

Мышьяк As — серебристо-серый хрупкий металлоид, ρ = 5,73 г/см³, возгоняется при 887°C. Полупроводник. Ядовит в любой форме и степени окисления.

Сурьма Sb — серебристо-белый блестящий хрупкий металлоид, Tпл = 630,6°C, ρ = 6,70 г/см³. Полупроводник с ромбоэдрической слоистой структурой.

Висмут Bi — серебристо-белый металл с розоватым оттенком, хрупкий. Tпл = 271,4°C, ρ = 9,78 г/см³. При затвердевании расширяется. Наибольший диамагнетизм среди всех элементов. Слабо радиоактивен (T₁/₂ ≈ 2·10¹⁹ лет).

4.2. Химические свойства

As, Sb и Bi проявляют степени окисления −3, +3 и +5. Устойчивость +5 убывает от As к Bi: у висмута Bi(+3) намного устойчивее Bi(+5). Оксиды E₂O₃ у As и Sb амфотерны с преобладанием кислотного характера, у Bi — с преобладанием основного. Характерная реакция — гидролиз галогенидов с образованием оксогалогенидов:

SbCl₃ + H₂O → SbOCl↓(белый) + 2HCl

BiCl₃ + H₂O → BiOCl↓(белый перламутровый) + 2HCl

Висмутат натрия — сильнейший окислитель; используется как качественная реакция на Mn²⁺:

2Mn²⁺ + 5NaBiO₃ + 14H⁺ → 2MnO₄⁻ + 5Bi³⁺ + 5Na⁺ + 7H₂O (малиновое окрашивание)

4.3. Применение

Мышьяк — полупроводник GaAs (лазерные диоды, быстродействующие транзисторы); легирование свинца в аккумуляторах.

Сурьма — сплавы со свинцом (аккумуляторные пластины, типографский металл); антипирены (Sb₂O₃ + галогенированные полимеры).

Висмут — легкоплавкие сплавы (сплав Вуда, Tпл = 68,7°C — тепловые предохранители); субсалицилат висмута (пепто-бисмол — противодиарейное, активен против H. pylori); BiOCl — перламутровый пигмент в косметике.
