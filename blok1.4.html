4. Закон постоянства состава. Относительные массы. Массовая доля

4.1. Закон постоянства состава

Закон постоянства состава (Ж. Л. Пруст, 1801–1808): каждое индивидуальное химическое соединение молекулярного строения имеет строго постоянный качественный и количественный состав, независимо от способа его получения и источника. Закон подтверждён лавуазьевской традицией аналитической химии и является следствием дальтоновской атомно-молекулярной теории.

Бертоллиды (нестехиометрические соединения) — вещества немолекулярного строения с переменным составом: TiOₓ (x = 0,7–1,3), FeₓO (x = 0,85–0,95), карбиды, нитриды, оксиды переходных металлов. В кристаллических решётках таких соединений часть узлов вакантна или замещена, отсюда — нецелые соотношения атомов. Закон Пруста к ним неприменим. Термин «бертоллиды» введён в противовес «дальтонидам» — соединениям с постоянным составом.

Закон кратных отношений (Дж. Дальтон, 1803): если два элемента образуют несколько соединений, то массы одного из них, приходящиеся на одинаковую массу другого, относятся как небольшие целые числа. Например, оксиды азота: в N₂O, NO, N₂O₃, NO₂, N₂O₅ массы кислорода, приходящиеся на 14 г азота, относятся как 1:2:3:4:5.

4.2. Относительные атомная и молекулярная массы

Абсолютные массы атомов и молекул ничтожно малы (масса протона m_p = 1,67262·10⁻²⁷ кг). Для практических расчётов используют относительные безразмерные величины, введённые через единицу атомной массы (а.е.м., u, Да): 1 а.е.м. = 1/12 · m(¹²C) = 1,66054·10⁻²⁷ кг.

Относительная атомная масса Ar — отношение средней массы атомов данного элемента (усреднённой по природным изотопным распространённостям) к 1 а.е.м. Значения Ar приведены в Периодической таблице. Ar — безразмерна, но численно равна молярной массе в г/моль.

Относительная молекулярная масса Mr = ΣAr_i · n_i — сумма относительных атомных масс с учётом числа атомов каждого элемента в формульной единице. Для ионных соединений корректнее говорить об относительной формульной массе. Примеры: Mr(H₂SO₄) = 2·1 + 32 + 4·16 = 98; Mr(Ca₃(PO₄)₂) = 3·40 + 2·(31 + 4·16) = 120 + 190 = 310.

4.3. Массовая доля элемента. Нахождение формулы по составу

Массовая доля элемента Э в соединении: ω(Э) = n · Ar(Э) / Mr, где n — число атомов элемента в формуле. Выражается в долях единицы (от 0 до 1) или в процентах (от 0 до 100%). Сумма массовых долей всех элементов = 1 (100%).

Пример. Для H₂SO₄ (Mr = 98): ω(H) = 2/98 = 0,0204 (2,04%); ω(S) = 32/98 = 0,3265 (32,65%); ω(O) = 64/98 = 0,6531 (65,31%). Сумма: 2,04 + 32,65 + 65,31 = 100%.

Нахождение эмпирической формулы по массовым долям: мольные доли каждого элемента n_i = ω_i / Ar_i; делят все на наименьшее значение, округляют до целых (или умножают на подходящее число). Пример: вещество содержит 40,0% C, 6,7% H, 53,3% O. n(C) = 40/12 = 3,33; n(H) = 6,7/1 = 6,7; n(O) = 53,3/16 = 3,33. Делим на 3,33: C:H:O = 1:2:1. Эмпирическая формула — CH₂O. Зная Mr, находим молекулярную: если Mr = 180, то n = 180/30 = 6, формула — C₆H₁₂O₆.

При вычислениях допускается округление Ar до целых значений (H = 1; C = 12; N = 14; O = 16; Na = 23; Mg = 24; Al = 27; S = 32; Cl = 35,5; K = 39; Ca = 40; Fe = 56; Cu = 64; Zn = 65; Br = 80; I = 127; Ba = 137), если в условии не оговорено иное. Исключение: Cl = 35,5 при точных расчётах, поскольку природный хлор содержит два изотопа в существенной пропорции.
