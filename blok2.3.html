Бинарные соединения

Бинарные соединения — вещества, молекулы или формульные единицы которых образованы атомами ровно двух химических элементов. Общая формула записывается как EₓYᵧ, где E и Y — символы элементов, x и y — стехиометрические коэффициенты. К бинарным соединениям относятся оксиды, гидриды, галогениды, сульфиды, нитриды, карбиды, фосфиды, силициды и ряд других классов. Их изучение составляет основу неорганической химии, поскольку именно бинарные соединения являются исходными продуктами для синтеза более сложных веществ.

Карбиды

1. Общая характеристика и классификация

Карбиды — бинарные соединения элементов с углеродом, в которых углерод является более электроотрицательным компонентом (или формально принят за таковой). Углерод имеет разные степени окисления в зависимости от типа карбида. По характеру химической связи и отношению к воде карбиды делят на три группы: ионные (солеподобные), ковалентные и металлоподобные.

Ионные (солеподобные) карбиды содержат ионы C⁴⁻ (метаниды — карбиды с отдельными ионами C), C₂²⁻ (ацетилениды — содержат «палочку» C≡C) или C₃⁴⁻ (аллилиды). Ковалентные карбиды — SiC (карборунд), B₄C — реальная ковалентная связь, высокая твёрдость и химическая инертность. Металлоподобные карбиды — TiC, WC, Cr₃C₂ — атомы C в межузлиях металлической решётки; проводники, твёрдые, тугоплавкие.

2. Ацетилениды (карбиды с C₂²⁻)

Ацетилениды содержат ион C₂²⁻, в котором между атомами углерода тройная связь (СО(C) = −1). При гидролизе дают ацетилен C₂H₂.

CaC₂ + 2H₂O → Ca(OH)₂ + C₂H₂↑ (промышленный способ получения ацетилена)

Na₂C₂ + 2H₂O → 2NaOH + C₂H₂↑

Ag₂C₂ + 2HCl → 2AgCl + C₂H₂↑ (качественная реакция на ацетилениды)

Карбид кальция CaC₂ — серое кристаллическое вещество, Tпл = 2160°C, ρ = 2,22 г/см³. На воздухе медленно разлагается влагой. Технический карбид кальция тёмно-серого цвета из-за примесей.

Получение CaC₂:

CaO + 3C → CaC₂ + CO↑ (T = 1900–2100°C, в электродуговых печах)

Химические свойства CaC₂:

CaC₂ + 2H₂O → Ca(OH)₂ + C₂H₂↑

CaC₂ + N₂ → CaCN₂ + C (цианамид кальция, T = 1100°C — удобрение)

CaC₂ + 5O₂ → 2CaO + 4CO₂ (горение)

2CaC₂ + 5O₂ → 2CaO + 4CO₂ (в избытке O₂)

Применение CaC₂: производство ацетилена для ацетиленовой сварки и резки металлов; синтез органических соединений (ацетальдегид, уксусная кислота, каучук через ацетилен); производство цианамида кальция CaCN₂ (удобрение, гербицид).

3. Метаниды (карбиды с C⁴⁻)

Метаниды содержат отдельные ионы C⁴⁻ (СО(C) = −4). При гидролизе дают метан CH₄.

Al₄C₃ + 12H₂O → 4Al(OH)₃ + 3CH₄↑ (гидролиз карбида алюминия)

Al₄C₃ + 12HCl → 4AlCl₃ + 3CH₄↑ (с кислотой)

Be₂C + 4H₂O → 2Be(OH)₂ + CH₄↑

Карбид алюминия Al₄C₃ — жёлтые кристаллы, Tпл = 2100°C. Получение: 4Al + 3C → Al₄C₃ (T > 1000°C). Применяется редко — легко гидролизуется.

Al₄C₃ — один из немногих метанидов, что делает его уникальным среди карбидов алюминия. Для сравнения: карбид кальция CaC₂ — ацетиленид (даёт C₂H₂), а не метанид. Тип карбида определяет продукт гидролиза — важная тема для заданий ОГЭ/ЕГЭ.

4. Карбид кремния SiC (карборунд)

Физические свойства: бесцветные (чистый) или зелёные/чёрные (технический) кристаллы, Tпл > 2700°C (разложение), ρ = 3,22 г/см³, твёрдость 9–9,5 по Моосу. Ковалентная атомная решётка (аналог алмаза с чередующимися Si и C). Полупроводник с широкой запрещённой зоной (2,3–3,3 эВ в зависимости от политипа).

Химические свойства: химически инертен. Не реагирует с кислотами, с HF и расплавленными щелочами реагирует лишь при высоких температурах. Реагирует с кислородом при нагревании:

SiC + 2O₂ → SiO₂ + CO₂ (T > 1000°C)

SiC + 2NaOH(расплав) → Na₂SiO₃ + C + H₂↑ (при высоких температурах)

Получение:

SiO₂ + 3C → SiC + 2CO↑ (процесс Ачесона, T = 2000°C, в электрических печах)

Применение: абразив (шлифовальные круги, наждачная бумага — «электрокорунд»); огнеупорная керамика (Tпл выше, чем у Al₂O₃); нагревательные элементы для высокотемпературных печей; силовые полупроводниковые приборы (диоды, транзисторы для работы при высоких температурах и напряжениях); бронежилеты (плитки из SiC — лёгкие и твёрдые).

5. Металлоподобные карбиды — WC, TiC

Карбид вольфрама WC — серые кристаллы, Tпл = 2870°C, твёрдость 9–9,5 по Моосу, высокая теплопроводность. Основа твёрдых сплавов (победит: WC + Co) — применяется для изготовления режущих инструментов, буровых коронок. Карбид титана TiC — Tпл = 3140°C, твёрдость ≈ 9,5 по Моосу. Применяется как покрытие режущего инструмента и в составе твёрдых сплавов.

6. Получение карбидов (общие методы)

Ca + 2C → CaC₂ (прямой синтез при нагревании, нечасто — обычно из CaO и C)

4Al + 3C → Al₄C₃ (T > 1000°C)

SiO₂ + 3C → SiC + 2CO↑ (T = 2000°C)

WO₃ + 4C → WC + 3CO↑ (T > 1500°C)

TiO₂ + 3C → TiC + 2CO↑ (T > 1600°C)

B₄C: B₂O₃ + 7C → B₄C + 6CO↑ (T = 2200°C)
