Бинарные соединения

Бинарные соединения — вещества, молекулы или формульные единицы которых образованы атомами ровно двух химических элементов. Общая формула записывается как EₓYᵧ, где E и Y — символы элементов, x и y — стехиометрические коэффициенты. К бинарным соединениям относятся оксиды, гидриды, галогениды, сульфиды, нитриды, карбиды, фосфиды, силициды и ряд других классов. Их изучение составляет основу неорганической химии, поскольку именно бинарные соединения являются исходными продуктами для синтеза более сложных веществ.

Галогениды

1.1. Общая характеристика и классификация

Галогениды — бинарные соединения элементов с галогенами (F, Cl, Br, I), в которых галоген имеет степень окисления −1. Общая формула: MeXₙ (для металлов) или EXₙ (для неметаллов), где X — атом галогена, n — валентность элемента. Галогениды — один из самых обширных классов неорганических соединений: практически каждый элемент образует хотя бы один галогенид.

По типу химической связи галогениды делятся на ионные и ковалентные. Ионные галогениды — соединения активных металлов (щелочных, щелочноземельных) с хлором, бромом и йодом: NaCl, KBr, CaF₂, MgCl₂. Характеризуются высокими температурами плавления, электропроводностью в расплаве и растворе, кристаллической ионной решёткой. Ковалентные галогениды — соединения неметаллов и металлов в высоких степенях окисления: PCl₅, SF₆, TiCl₄, AlCl₃, FeCl₃. Характеризуются молекулярным строением, относительно низкими температурами кипения и выраженной склонностью к гидролизу. Граница между типами размыта: AlCl₃, например, в твёрдом состоянии проявляет ионный характер, а в газовой фазе существует как ковалентный димер Al₂Cl₆.

Реакционная способность галогенов убывает в ряду F₂ > Cl₂ > Br₂ > I₂. Каждый галоген вытесняет последующие из растворов их солей: Cl₂ + 2KBr → 2KCl + Br₂; Cl₂ + 2KI → 2KCl + I₂; Br₂ + 2KI → 2KBr + I₂. Обратные реакции не идут — это основа качественного и количественного анализа галогенидов.

1.2. Химические свойства галогенидов металлов

Реакция с нитратом серебра (качественная реакция на галогенид-ион, проводят в присутствии HNO₃ разб.):

Ag⁺ + Cl⁻ → AgCl↓ (белый творожистый осадок, растворяется в NH₃·H₂O)

Ag⁺ + Br⁻ → AgBr↓ (светло-жёлтый осадок, слабо растворяется в NH₃·H₂O)

Ag⁺ + I⁻ → AgI↓ (жёлтый осадок, не растворяется в NH₃·H₂O)

Реакцию проводят в присутствии HNO₃ разб. — чтобы исключить осаждение других анионов (CO₃²⁻, SO₄²⁻, PO₄³⁻), которые тоже дают осадки с Ag⁺. F⁻ определяют отдельно: AgF растворим, поэтому на F⁻ используют Ca²⁺ — CaF₂ даёт белый осадок.

Гидролиз ковалентных галогенидов — галогениды неметаллов и металлов в высоких степенях окисления интенсивно гидролизуются водой:

PCl₅ + 4H₂O → H₃PO₄ + 5HCl

PCl₃ + 3H₂O → H₃PO₃ + 3HCl

AlCl₃ + 3H₂O ⇌ Al(OH)₃ + 3HCl (ступенчатый гидролиз, pH раствора 3–4)

SiCl₄ + 4H₂O → Si(OH)₄ + 4HCl (полный гидролиз)

TiCl₄ + 4H₂O → TiO₂·H₂O↓ + 4HCl

Ионные галогениды (NaCl, KCl, CaCl₂) гидролизу не подвергаются — это соли сильных кислот и сильных оснований.

Вытеснение галогенидов из солей более активными галогенами:

Cl₂ + 2NaBr → 2NaCl + Br₂

Br₂ + 2NaI → 2NaBr + I₂

Cl₂ + 2KI → 2KCl + I₂

Реакции ионных галогенидов с концентрированной H₂SO₄:

NaCl + H₂SO₄(конц.) → NaHSO₄ + HCl↑ (умеренный нагрев)

2NaCl + H₂SO₄(конц.) → Na₂SO₄ + 2HCl↑ (сильный нагрев)

2NaBr + 2H₂SO₄(конц.) → Na₂SO₄ + Br₂↑ + SO₂↑ + 2H₂O (бромид окисляется)

8NaI + 5H₂SO₄(конц.) → 4Na₂SO₄ + 4I₂↑ + H₂S↑ + 4H₂O (иодид — сильный восстановитель)

NaBr и NaI нельзя получить действием конц. H₂SO₄ — она их окисляет. Для получения HBr и HI из солей применяют H₃PO₄ (не является окислителем): NaBr + H₃PO₄ → NaH₂PO₄ + HBr↑. Это принципиальное отличие от получения HCl, где конц. H₂SO₄ работает нормально.

Реакция фторидов с SiO₂ (специфика фторидов):

CaF₂ + SiO₂ → CaSiO₃ + SiF₄↑ (при сплавлении)

Na₂SiO₃ + 6NaF + 3SO₃ → 6NaF... (нет — правильно:)

4NaF + SiO₂ + 2H₂SO₄ → SiF₄↑ + 2Na₂SO₄ + 2H₂O

Реакция хлоридов с окислителями:

2FeCl₂ + Cl₂ → 2FeCl₃ (окисление Fe²⁺ до Fe³⁺)

2FeCl₃ + Cu → 2FeCl₂ + CuCl₂ (FeCl₃ — окислитель, используется при травлении плат)

2FeCl₃ + Fe → 3FeCl₂ (Fe восстанавливает Fe³⁺ до Fe²⁺)

2KMnO₄ + 16HCl(конц.) → 2KCl + 2MnCl₂ + 5Cl₂↑ + 8H₂O (HCl — восстановитель)

1.3. Получение галогенидов

Прямой синтез из элементов:

2Na + Cl₂ → 2NaCl

2Fe + 3Cl₂ → 2FeCl₃ (хлор окисляет Fe до +3 — в отличие от серы и иода, которые дают FeS и FeI₂)

Cu + Cl₂ → CuCl₂

Sn + 2Cl₂ → SnCl₄ (до высшей валентности при избытке Cl₂)

2P + 3Cl₂(недост.) → 2PCl₃; 2P + 5Cl₂(изб.) → 2PCl₅

Реакция металлов с галогеноводородными кислотами:

Zn + 2HCl → ZnCl₂ + H₂↑

Fe + 2HCl → FeCl₂ + H₂↑

Mg + 2HBr → MgBr₂ + H₂↑

Реакция оксидов и гидроксидов с галогеноводородными кислотами:

CuO + 2HCl → CuCl₂ + H₂O

Al₂O₃ + 6HCl → 2AlCl₃ + 3H₂O

NaOH + HCl → NaCl + H₂O

Реакции обмена (осаждение нерастворимых галогенидов):

AgNO₃ + NaCl → AgCl↓ + NaNO₃

Pb(NO₃)₂ + 2NaCl → PbCl₂↓ + 2NaNO₃

Взаимодействие солей с галогенами (вытеснение):

2KBr + Cl₂ → 2KCl + Br₂

2KI + Br₂ → 2KBr + I₂

1.4. Физические свойства и применение важнейших галогенидов

Хлорид натрия NaCl — белое кристаллическое вещество, Tпл = 801°C, ρ = 2,165 г/см³. Хорошо растворим в воде (35,9 г/100 мл при 20°C), растворимость мало зависит от температуры. Ионная кристаллическая решётка типа ГЦК. В водном растворе полностью диссоциирует. Гидролизу не подвергается. Применение: пищевая промышленность, производство Na, NaOH, Cl₂, HCl, Na₂CO₃ электролизом.

Хлорид кальция CaCl₂ — белое вещество, Tпл = 772°C. Очень хорошо растворим (74,5 г/100 мл при 20°C), образует кристаллогидрат CaCl₂·6H₂O. Сильно гигроскопичен. Применение: осушитель (нельзя применять для сушки NH₃ и аминов — образует аддукты CaCl₂·8NH₃), антиобледенитель дорог (снижает температуру замерзания воды до −35°C), производство Ca.

Фторид кальция CaF₂ (флюорит, плавиковый шпат) — белое нерастворимое вещество, Tпл = 1418°C, ρ = 3,18 г/см³. Важнейший природный минерал. Применение: производство HF и фтора, добавка при выплавке чугуна (флюс — снижает вязкость шлака), производство оптического стекла и призм (CaF₂ прозрачен в ИК и УФ диапазонах).

Хлорид алюминия AlCl₃ — белое вещество, возгоняется при 178°C (не плавится при атм. давлении). В газовой фазе — ковалентный димер Al₂Cl₆. Сильный акцептор Льюиса. Бурно гидролизуется на воздухе (дымит). Применение: катализатор реакций Фриделя–Крафтса в органическом синтезе.

Хлорид железа(III) FeCl₃ — тёмно-коричневые кристаллы, Tпл = 307°C. Хорошо растворим в воде; раствор кислый (гидролиз). Является окислителем — применяется при травлении медных печатных плат: 2FeCl₃ + Cu → 2FeCl₂ + CuCl₂.

Фторид натрия NaF — белое вещество, Tпл = 993°C. Токсичен. Применение: консервант в производстве клея и красок, добавка в зубные пасты (профилактика кариеса), компонент флюсов при сварке.

Хлорид титана(IV) TiCl₄ — бесцветная жидкость, Tкип = 136°C. Дымит на воздухе (гидролиз с образованием TiO₂·H₂O и HCl). Применение: промежуточный продукт в производстве Ti (восстановление магнием: TiCl₄ + 2Mg → Ti + 2MgCl₂ — метод Кролля), катализатор полимеризации (Циглера–Натта).
