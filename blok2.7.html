Гидроксиды

1. Определение и классификация

Гидроксиды — соединения оксидов с водой, содержащие одну или несколько гидроксогрупп OH. В зависимости от характера диссоциации и отношения к кислотам и щелочам гидроксиды делятся на три группы: основные, амфотерные и кислотные.

Основные гидроксиды (основания) диссоциируют с отщеплением OH⁻: Me(OH)ₙ → Meⁿ⁺ + nOH⁻. Образованы металлами в СО +1 и +2 (кроме амфотерных исключений). Реагируют с кислотами, не реагируют со щелочами. Амфотерные гидроксиды диссоциируют по кислотному и по основному типу в зависимости от условий. Образованы металлами в СО +2 (Be, Zn, Sn, Pb — те же исключения что и у оксидов) и в СО +3, +4. Реагируют и с кислотами, и со щелочами. Кислотные гидроксиды диссоциируют с отщеплением H⁺. Образованы неметаллами и металлами в высоких степенях окисления (+5 и выше). Подробно рассмотрены в разделе «Кислоты».

Принадлежность гидроксида к той или иной группе определяется теми же правилами, что и у оксидов: металлы в СО +1 и +2 — основные (кроме Be(OH)₂, Zn(OH)₂, Sn(OH)₂, Pb(OH)₂); металлы в СО +3 и +4, а также перечисленные исключения — амфотерные; неметаллы и металлы в СО +5 и выше — кислотные (H₂SO₄, HNO₃, H₃PO₄ и т.д.).

2. Основные гидроксиды (основания)

2.1. Общая характеристика. Щёлочи и нерастворимые основания

Основные гидроксиды делятся на два принципиально различных подкласса по растворимости. Щёлочи — растворимые основания: NaOH, KOH, LiOH, Ba(OH)₂, Ca(OH)₂ (малорастворим, но считается щёлочью), RbOH, CsOH. Нерастворимые основания — практически нерастворимы в воде: Cu(OH)₂, Fe(OH)₂, Fe(OH)₃, Mg(OH)₂, Ni(OH)₂, Co(OH)₂, Mn(OH)₂ и большинство гидроксидов переходных металлов.

Сила оснований возрастает в ряду щелочных металлов Li → Cs. Среди щелочноземельных: Mg(OH)₂ — нерастворимое слабое основание; Ca(OH)₂ — малорастворимое сильное основание; Ba(OH)₂ — хорошо растворимая сильная щёлочь. Растворимость гидроксидов в группе II A закономерно возрастает от Mg к Ba.

2.2. Физические свойства

Щёлочи — белые твёрдые кристаллические вещества при обычных условиях, все сильно гигроскопичны и поглощают CO₂ из воздуха. NaOH (едкий натр) — белые непрозрачные пластинки или гранулы, Tпл = 318°C, ρ = 2,13 г/см³, растворимость 109 г/100 мл при 20°C, растворение в воде сопровождается сильным выделением тепла. KOH (едкое кали) — белые гигроскопичные кристаллы, Tпл = 360°C, растворимость 112 г/100 мл при 20°C. Ca(OH)₂ (гашёная известь) — белый порошок, растворимость всего 0,17 г/100 мл при 20°C; растворимость аномально убывает с ростом температуры. Ba(OH)₂ — белые кристаллы, Tпл = 408°C, растворимость 3,7 г/100 мл при 20°C.

Все щёлочи вызывают тяжёлые химические ожоги — разрушают белки кожи и омыляют жиры. Особенно опасны при попадании в глаза.

Нерастворимые основания — рыхлые осадки характерной окраски: Cu(OH)₂ — голубой; Fe(OH)₂ — белый, быстро зеленеет и буреет на воздухе; Fe(OH)₃ — бурый; Ni(OH)₂ — зелёный; Co(OH)₂ — розовый; Mn(OH)₂ — белый, темнеет на воздухе; Cr(OH)₃ — серо-зелёный.

2.3. Химические свойства щелочей

Реакция с кислотами (нейтрализация):

NaOH + HCl → NaCl + H₂O

2NaOH + H₂SO₄ → Na₂SO₄ + 2H₂O

3KOH + H₃PO₄ → K₃PO₄ + 3H₂O

NaOH + H₃PO₄(недост.) → NaH₂PO₄ + H₂O (кислая соль при недостатке щёлочи)

2NaOH + H₃PO₄ → Na₂HPO₄ + 2H₂O (при соотношении 2:1)

Реакция с кислотными оксидами:

2NaOH + CO₂ → Na₂CO₃ + H₂O (при избытке NaOH)

NaOH + CO₂(изб.) → NaHCO₃ (при избытке CO₂)

2NaOH + SO₂ → Na₂SO₃ + H₂O

2KOH + SO₃ → K₂SO₄ + H₂O

2NaOH + SiO₂ → Na₂SiO₃ + H₂O (при сплавлении или кипячении с конц. раствором)

Реакция с амфотерными оксидами и гидроксидами:

2NaOH + Al₂O₃ → 2NaAlO₂ + H₂O (при сплавлении)

2NaOH + Al₂O₃ + 3H₂O → 2Na[Al(OH)₄] (в водном растворе)

2NaOH + Zn(OH)₂ → Na₂[Zn(OH)₄]

2KOH + Be(OH)₂ → K₂[Be(OH)₄]

Реакция с амфотерными металлами:

2Al + 2NaOH + 6H₂O → 2Na[Al(OH)₄] + 3H₂↑

Zn + 2NaOH + 2H₂O → Na₂[Zn(OH)₄] + H₂↑

Be + 2NaOH + 2H₂O → Na₂[Be(OH)₄] + H₂↑

Si + 2NaOH + H₂O → Na₂SiO₃ + 2H₂↑

Реакция с солями (ионный обмен — идёт если выпадает осадок, выделяется газ или образуется слабый электролит):

2NaOH + CuSO₄ → Cu(OH)₂↓ + Na₂SO₄

3NaOH + FeCl₃ → Fe(OH)₃↓ + 3NaCl

2NaOH + MgCl₂ → Mg(OH)₂↓ + 2NaCl

4NaOH(изб.) + ZnSO₄ → Na₂[Zn(OH)₄] + Na₂SO₄ (при избытке NaOH осадок Zn(OH)₂ растворяется)

Ba(OH)₂ + Na₂SO₄ → BaSO₄↓ + 2NaOH

Ba(OH)₂ + 2NH₄Cl → BaCl₂ + 2NH₃↑ + 2H₂O

Реакция с неметаллами:

Cl₂ + 2NaOH(хол.) → NaCl + NaClO + H₂O

3Cl₂ + 6NaOH(гор.) → 5NaCl + NaClO₃ + 3H₂O

S + 6NaOH(конц., гор.) → 2Na₂S + Na₂SO₃ + 3H₂O

P₄ + 3KOH + 3H₂O → 3KH₂PO₂ + PH₃↑

Термическое разложение — щёлочи щелочных металлов при нагревании не разлагаются. Ca(OH)₂ разлагается при сильном нагревании:

Ca(OH)₂ → CaO + H₂O (T > 580°C)

Щёлочи не реагируют с нерастворимыми солями слабых кислот (CaCO₃, BaSO₄ и т.п.). Реакция обмена идёт только если в результате образуется осадок, выделяется газ или слабый электролит — иначе реакция в растворе не протекает.

2.4. Химические свойства нерастворимых оснований

Реакция с кислотами:

Cu(OH)₂ + H₂SO₄ → CuSO₄ + 2H₂O

Fe(OH)₃ + 3HCl → FeCl₃ + 3H₂O

Mg(OH)₂ + 2HNO₃ → Mg(NO₃)₂ + 2H₂O

Со щелочами нерастворимые основания не реагируют — это принципиальный признак, отличающий их от амфотерных гидроксидов.

Термическое разложение при нагревании:

Cu(OH)₂ → CuO + H₂O (T ≈ 80°C)

2Fe(OH)₃ → Fe₂O₃ + 3H₂O (при прокаливании)

Mg(OH)₂ → MgO + H₂O (T ≈ 350°C)

Co(OH)₂ → CoO + H₂O (T > 160°C)

Окисление на воздухе — характерно для гидроксидов металлов в низших степенях окисления:

4Fe(OH)₂ + O₂ + 2H₂O → 4Fe(OH)₃ (побурение белого осадка — качественная реакция на Fe²⁺)

4Mn(OH)₂ + O₂ → 4MnO(OH) + 2H₂O (темнение белого осадка)

2.5. Получение оснований

Получение щелочей:

2Na + 2H₂O → 2NaOH + H₂↑

Ca + 2H₂O → Ca(OH)₂ + H₂↑

Na₂O + H₂O → 2NaOH

BaO + H₂O → Ba(OH)₂

2NaCl + 2H₂O → 2NaOH + Cl₂↑ + H₂↑ (электролиз раствора NaCl — основной промышленный способ)

Na₂CO₃ + Ca(OH)₂ → 2NaOH + CaCO₃↓ (каустификация соды)

CaO + H₂O → Ca(OH)₂ (гашение извести)

Получение нерастворимых оснований — только осаждением из растворов солей:

CuSO₄ + 2NaOH → Cu(OH)₂↓ + Na₂SO₄

FeCl₃ + 3NaOH → Fe(OH)₃↓ + 3NaCl

FeCl₂ + 2NaOH → Fe(OH)₂↓ + 2NaCl

MgCl₂ + 2NaOH → Mg(OH)₂↓ + 2NaCl

AlCl₃ + 3NH₃·H₂O → Al(OH)₃↓ + 3NH₄Cl (аммиак вместо NaOH — чтобы не растворить амфотерный осадок)

Нерастворимые основания нельзя получить реакцией оксида с водой — они с водой не реагируют. Единственный путь — осаждение из раствора соответствующей соли. При этом важно не переборщить со щёлочью: амфотерные гидроксиды Al(OH)₃, Zn(OH)₂, Cr(OH)₃ при избытке NaOH растворяются.

2.6. Применение

NaOH — одно из крупнотоннажнейших химических производств мира. Применяется в производстве мыла и моющих средств (омыление жиров); производстве вискозы, целлюлозы и бумаги; нефтепереработке (очистка нефтепродуктов от серосодержащих примесей); производстве Al₂O₃ из бокситов (метод Байера: Al(OH)₃ растворяют в NaOH); синтезе красителей, фармацевтических препаратов; в качестве электролита в щелочных аккумуляторах.

KOH — производство жидкого мыла (калиевые мыла мягче натриевых); электролит в щелочных топливных элементах и аккумуляторах; производство K₂CO₃ и других соединений калия.

Ca(OH)₂ (гашёная известь) — строительство (штукатурка, известковый раствор); нейтрализация кислотных почв в сельском хозяйстве; производство хлорной извести Ca(ClO)₂; очистка сточных вод; производство соды аммиачным методом (Сольвея).

Mg(OH)₂ — медицина (антацид «молоко магнезии», нейтрализует избыточную кислоту в желудке); антипирен в полимерных материалах (при нагревании разлагается с поглощением тепла и выделением воды).

Fe(OH)₃ — адсорбент при отравлении соединениями As и некоторыми тяжёлыми металлами; пигмент; используется в аналитической химии.

Cu(OH)₂ — фунгицид в сельском хозяйстве (бордоская жидкость — смесь Cu(OH)₂ и Ca(OH)₂); реагент Фелинга для определения альдегидов и моносахаридов.

3. Амфотерные гидроксиды

3.1. Общая характеристика

Амфотерные гидроксиды реагируют и с кислотами (проявляя основные свойства), и со щелочами (проявляя кислотные свойства). Двойственность поведения объясняется промежуточным значением электроотрицательности металла: связь O–H полярна настолько, что может разрываться как по основному (→ OH⁻), так и по кислотному (→ H⁺) типу в зависимости от реагента.

Важнейшие представители: Al(OH)₃, Zn(OH)₂, Be(OH)₂, Cr(OH)₃, Sn(OH)₂, Pb(OH)₂, SnO(OH)₂ (метаоловянная кислота). Fe(OH)₃ проявляет очень слабые амфотерные свойства (реагирует со щелочами только при сплавлении).

3.2. Физические свойства

Все амфотерные гидроксиды — нерастворимые в воде осадки. Al(OH)₃ — белый студенистый осадок, разлагается при T > 300°C: 2Al(OH)₃ → Al₂O₃ + 3H₂O. Zn(OH)₂ — белый осадок, разлагается при T ≈ 200°C: Zn(OH)₂ → ZnO + H₂O. Be(OH)₂ — белый осадок. Cr(OH)₃ — серо-зелёный осадок, разлагается при прокаливании: 2Cr(OH)₃ → Cr₂O₃ + 3H₂O. Pb(OH)₂ — белый осадок. Все термически нестойки и разлагаются при нагревании значительно легче, чем гидроксиды щелочных металлов.

3.3. Химические свойства

Реакция с кислотами (основные свойства):

Al(OH)₃ + 3HCl → AlCl₃ + 3H₂O

Al(OH)₃ + 3HNO₃ → Al(NO₃)₃ + 3H₂O

Zn(OH)₂ + H₂SO₄ → ZnSO₄ + 2H₂O

Be(OH)₂ + 2HCl → BeCl₂ + 2H₂O

Cr(OH)₃ + 3HCl → CrCl₃ + 3H₂O

Pb(OH)₂ + 2HNO₃ → Pb(NO₃)₂ + 2H₂O

Реакция со щелочами (кислотные свойства):

Al(OH)₃ + NaOH → Na[Al(OH)₄] (в растворе — тетрагидроксоалюминат)

Al(OH)₃ + NaOH → NaAlO₂ + 2H₂O (при сплавлении — метаалюминат)

Zn(OH)₂ + 2NaOH → Na₂[Zn(OH)₄] (в растворе)

Zn(OH)₂ + 2NaOH → Na₂ZnO₂ + 2H₂O (при сплавлении)

Be(OH)₂ + 2NaOH → Na₂[Be(OH)₄]

Cr(OH)₃ + NaOH → Na[Cr(OH)₄] (в растворе — слабее, чем Al и Zn)

Cr(OH)₃ + NaOH → NaCrO₂ + 2H₂O (при сплавлении)

Pb(OH)₂ + 2NaOH → Na₂[Pb(OH)₄]

Термическое разложение:

2Al(OH)₃ → Al₂O₃ + 3H₂O (T > 300°C)

Zn(OH)₂ → ZnO + H₂O (T ≈ 200°C)

2Cr(OH)₃ → Cr₂O₃ + 3H₂O (при прокаливании)

Al(OH)₃ — самый важный амфотерный гидроксид в рамках ГИА. Ключевые моменты: реагирует и с кислотами, и с щелочами; при избытке щёлочи осадок полностью растворяется; для осаждения Al(OH)₃ из раствора аммиак предпочтительнее NaOH, так как NH₃ — слабое основание и не растворит осадок. Cr(OH)₃ ведёт себя аналогично Al(OH)₃, но реагирует со щелочами в растворе значительно хуже.

3.4. Получение амфотерных гидроксидов

Осаждение из растворов солей недостаточным количеством щёлочи или аммиаком:

AlCl₃ + 3NaOH(недост.) → Al(OH)₃↓ + 3NaCl

AlCl₃ + 3NH₃·H₂O → Al(OH)₃↓ + 3NH₄Cl (предпочтительный способ)

ZnSO₄ + 2NaOH(недост.) → Zn(OH)₂↓ + Na₂SO₄

CrCl₃ + 3NaOH(недост.) → Cr(OH)₃↓ + 3NaCl

Pb(NO₃)₂ + 2NaOH(недост.) → Pb(OH)₂↓ + 2NaNO₃

Реакция солей алюминия с раствором соды (частичный гидролиз):

AlCl₃ + 3Na₂CO₃ + 3H₂O → 2Al(OH)₃↓ + 3CO₂↑ + 6NaCl (совместный гидролиз)

3.5. Применение

Al(OH)₃ — антацид в медицине (препараты «Алмагель», «Маалокс» — нейтрализует избыточную HCl в желудке, не вызывая «кислотного рикошета»); антипирен и наполнитель в полимерных материалах (при нагревании разлагается с поглощением тепла); коагулянт при очистке воды (Al₂(SO₄)₃ гидролизуется с образованием Al(OH)₃, который сорбирует взвеси); производство Al₂O₃ и далее металлического Al.

Zn(OH)₂ — промежуточный продукт в производстве вискозы (ZnCl₂ применяется как растворитель целлюлозы); в медицине в составе антисептических и вяжущих мазей.

Cr(OH)₃ — промежуточный продукт при получении Cr₂O₃ и хромовых красителей; используется в кожевенной промышленности (хромовое дубление).

4. Кислотные гидроксиды

4.1. Общая характеристика

Кислотные гидроксиды — гидроксиды, диссоциирующие с отщеплением протона H⁺. Они являются кислотами в традиционном понимании: H₂SO₄, HNO₃, H₃PO₄, HClO₄, H₂SiO₃, HMnO₄ и многие другие. Образованы неметаллами (в любой степени окисления) и металлами в степенях окисления +5 и выше: CrO₃ → H₂CrO₄, Mn₂O₇ → HMnO₄, V₂O₅ → HVO₃.

Формально кислотный гидроксид — это то же самое вещество, которое в разделе о кислотных оксидах называлось «соответствующей кислотой»: SO₃ + H₂O → H₂SO₄, N₂O₅ + H₂O → 2HNO₃. Таким образом, кислотный гидроксид — это кислотный оксид, соединённый с водой.

Кислотные гидроксиды — это кислоты. Их классификация, физические и химические свойства, получение и применение подробно рассмотрены в разделе «Кислоты».
