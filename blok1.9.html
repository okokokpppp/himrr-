9. Химические реакции. Классификация. Термохимия. Окислительно-восстановительные реакции

9.1. Признаки и условия химических реакций

Химическая реакция — процесс превращения одних веществ (реагентов) в другие (продукты) с разрывом и образованием химических связей. Химический состав и строение реагентов изменяются, тогда как суммарное число атомов каждого элемента и их масса сохраняются (закон сохранения массы, Ломоносов–Лавуазье, 1748/1774). Признаки реакции: изменение цвета, образование осадка, выделение газа, изменение запаха, тепловой эффект (свечение, нагрев или охлаждение).

Условия протекания реакций: нагревание (повышение T), освещение (фотохимические реакции), давление, наличие катализатора, измельчение (для гетерогенных реакций), электрический ток (электролиз). Необходимое условие — контакт реагентов; достаточное — превышение энергии активации.

9.2. Классификация реакций

По числу и составу реагентов и продуктов: реакции соединения (A + B → AB, например 2Mg + O₂ → 2MgO); реакции разложения (AB → A + B, например 2H₂O → 2H₂ + O₂); реакции замещения (A + BC → AC + B, например Fe + CuSO₄ → FeSO₄ + Cu); реакции обмена (AB + CD → AD + CB, например NaCl + AgNO₃ → AgCl↓ + NaNO₃).

Классификация по числу веществ неточна: некоторые реакции сочетают признаки нескольких типов. Реакция диспропорционирования (самоокисление-самовосстановление): Cl₂ + 2NaOH → NaCl + NaClO + H₂O — один и тот же элемент одновременно окисляется и восстанавливается.

По тепловому эффекту: экзотермические (ΔH меньше 0, теплота выделяется: горение, нейтрализация, большинство реакций соединения) и эндотермические (ΔH > 0, теплота поглощается: разложение карбонатов, восстановление металлов углём, некоторые реакции с N₂). Тепловой эффект Q = −ΔH при постоянном давлении.

По обратимости: обратимые (⇌, устанавливается химическое равновесие) и необратимые (→, протекают практически до конца при образовании осадка, газа, слабого электролита или при очень высокой экзотермичности).

По фазовому составу: гомогенные (все участники в одной фазе, например реакции в растворах и газах) и гетерогенные (реагенты в разных фазах: горение угля C(тв) + O₂(г), растворение металлов в кислотах).

По участию катализатора: каталитические (MnO₂ как катализатор разложения H₂O₂) и некаталитические. Ферментативные реакции — частный случай каталитических (биологические катализаторы — ферменты, белковой природы).

9.3. Термохимия. Закон Гесса

Тепловой эффект реакции (энтальпия реакции ΔH) — теплота, поглощаемая или выделяемая в ходе реакции при постоянном давлении. Единица — кДж/моль. Стандартная энтальпия образования ΔH°f — тепловой эффект образования 1 моль вещества из простых веществ при стандартных условиях (25°C, 100 кПа). Для простых веществ в стандартном состоянии ΔH°f = 0.

Закон Гесса (1840): тепловой эффект химической реакции не зависит от пути её протекания, а определяется только начальным и конечным состоянием системы. Следствие: ΔH°реакции = Σ ΔH°f(продуктов) − Σ ΔH°f(реагентов) (с учётом стехиометрических коэффициентов). Закон позволяет рассчитывать тепловые эффекты реакций, которые нельзя провести непосредственно.

Термохимическое уравнение включает агрегатные состояния всех участников (обязательно!) и значение ΔH. Например: C(графит) + O₂(г) → CO₂(г), ΔH = −393,5 кДж/моль. Агрегатное состояние принципиально: ΔH°f(H₂O, г) = −241,8 кДж/моль, ΔH°f(H₂O, ж) = −285,8 кДж/моль; разница 44 кДж/моль — теплота конденсации пара.

9.4. Окислительно-восстановительные реакции (ОВР)

ОВР — реакции, сопровождающиеся изменением степеней окисления элементов вследствие переноса электронов от восстановителя к окислителю. Окисление — потеря электронов (степень окисления возрастает). Восстановление — приобретение электронов (степень окисления уменьшается). Окислитель — принимает электроны и восстанавливается. Восстановитель — отдаёт электроны и окисляется. Оба процесса взаимно сопряжены и невозможны один без другого.

Метод электронного баланса: составить уравнения полуреакций окисления и восстановления; уравнять числа электронов путём умножения полуреакций на соответствующие коэффициенты; подставить коэффициенты в исходное уравнение; проверить баланс атомов и зарядов.

Пример: MnO₄⁻ + Fe²⁺ → Mn²⁺ + Fe³⁺ (в кислой среде). Восстановление: MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O (Mn: +7→+2, 5e⁻). Окисление: Fe²⁺ − e⁻ → Fe³⁺ (Fe: +2→+3, 1e⁻). Множители: ×1 и ×5. Итог: MnO₄⁻ + 5Fe²⁺ + 8H⁺ → Mn²⁺ + 5Fe³⁺ + 4H₂O.

Влияние среды на ОВР с перманганатом: в кислой среде KMnO₄ восстанавливается до Mn²⁺ (бесцветный); в нейтральной или слабощелочной — до MnO₂ (бурый осадок, +4); в сильнощелочной — до MnO₄²⁻ (манганат, зелёный, +6). Аналогично для K₂Cr₂O₇: в кислой среде Cr₂O₇²⁻ (оранжевый) → Cr³⁺ (зелёный); в щелочной — CrO₄²⁻ → Cr(OH)₄⁻ → Cr(OH)₃.

Типичные окислители: KMnO₄, K₂Cr₂O₇, HNO₃ (конц. и разб.), H₂SO₄ (конц.), Cl₂, Br₂, O₂, O₃, H₂O₂ (в кислой среде), MnO₂, PbO₂, Na₂O₂. Типичные восстановители: металлы (в ряду напряжений — активные металлы — сильнейшие), H₂, C, CO, SO₂, H₂S, HI, HBr (последний — слабее HI), H₂O₂ (в щелочной среде), FeO, SnCl₂, органические вещества (глюкоза, формальдегид, муравьиная кислота).

Реакции диспропорционирования: один элемент одновременно выступает окислителем и восстановителем. Пример: 2H₂O₂ → 2H₂O + O₂ (O: −1 → −2 и −1 → 0); 3HNO₂ → HNO₃ + 2NO + H₂O (N: +3 → +5 и +3 → +2); Cl₂ + 2NaOH → NaCl + NaClO + H₂O (Cl: 0 → −1 и 0 → +1). Реакции конпропорционирования (сближение СО): S + H₂SO₄(конц) → SO₂; NO + NO₂ + H₂O → 2HNO₂.
